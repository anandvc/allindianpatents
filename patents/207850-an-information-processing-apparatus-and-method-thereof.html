DESCRIPTION Method and Apparatus for Controlling Reserved Recording of Television Broadcast Program and Recording Medium Having a Program Recorded Thoroon
Technical Field
This invention relates to an information processing method and apparatus and a medium. More particularly, it relates to an information processing method and apparatus and a medium in which recording can be reserved in a picture recording apparatus.
Background Art
The technology of having a tuner enclosed in a personal computer to receive signals of pictures and speech from a television broadcasting station, converting the received pictures and speech into pre-set digital data to record the converted data on a recording medium, such as a hard disc, and of reproducing the recorded data in case of necessity, is finding widespread application.
In such personal computer, setting a channel for reservation recording is extremely labor- and time-consuming.
Disclosure of the Invention
It is therefore an object of the present invention to enable reservation recording

to be set readily and speedily.
The present invention relates to an information processing apparatus comprising:
means for receiving a message requesting a recording reservation data file for controlling the recording of a broadcast signal including data for specifying a channel, data indicating the date and time for starting the recording and data indicating said date and time for terminating the recording from a television recording apparatus via the internet; and
transmission means for transmitting said recording reservation data file to said television recording apparatus via the internet, wherein said recording reservation data file is constituted by text data which can be written by using a word processor or a text editor.
The invention also relates to an information processing method comprising:
a step of receiving a message requesting a recording reservation data file for controlling the recording of a broadcast signal including data for specifying a channel, data indicating the date and time for starting the recording and data indicating said date and time for terminating the recording from a television apparatus via the internet; and
a step of transmitting said recording reservation data file to said television recording apparatus via the internet, wherein said recording reservation data file is constituted by text data which can be written by using a word processor or a text editor.
