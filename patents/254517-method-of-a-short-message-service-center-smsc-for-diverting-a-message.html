FIELD OF INVENTION
This invention relates, in general, to mobile communication and messaging in mobile communications. This invention further relates to where the messages expire in Short Message Service Center (SMSC) and the user loses some information. More particularly the present invention relates to Message Divert To Mail Id (Divert Messages to Mail ID when user is Unreachable or the validity period expires in the SMSC).
DESCRIPTION OF RELATED ART
In the present art, the option of forwarding SMS to email accounts is not available. If the validity period of the message expires in the SMSC, the message is deleted and an error message is sent to the user who has sent the message. Therefore, alternate destination sending is not possible.
LIMITATIONS
Currently, if a user moves to a non-coverage area, there is no provision to know about people who are trying to reach them. In other words the user is not connected to the external world.
If the user has switched off the mobile and the validity period of the message expires, he can still know the people who are trying to reach him through his mail id.

SUMMARY OF THE INVENTION
This invention proposes a method by which an SMS after being received at the SMSC and after expiry of the validity period is sent to the recipient's email ID instead of being simply deleted. In such a case, the recipient has a better chance of receiving the message even though his mobile may not be accessible during the validity period of the message.
According this invention explains a method of message diversion to mail ID comprising the steps of:
(a) receiving the message by a SMSC from a sender UE;
(b) checking whether the message is delivered by the said SMSC to a receiver
UE;
(c) delivering the message to the receiver UE if the said receiver UE is in a
condition to receive the said message;
(d) checking whether validity period of the said message has expired at the said
SMSC if the message is not delivered to the said receiver UE;
(e) checking whether email id corresponding to the said receiver UE is
registered if the said validity period has expired; and
(f) sending the said message to the said corresponding email id if email id
corresponding to the destination number is registered.
The said message is diverted to the corresponding email id in case when said

receiver UE is not reachable. The said message is diverted to the corresponding email id in case when said receiver UE is switched-off. The email id corresponding to the destination number is registered using a registration mechanism or by sending an SMS. A provision is given for the sender UE to specify the email id to which the said message is suppose to be sent through an Ul.

Explanation of the procedure when User mobile is switched off or not reachable
In this invention the User mobile not reachable or switched off are preliminary conditions among many others that can cause the message not getting delivered before its expiry.
This invention states if the message is not diverted to the User Mobile from SMSC due to reasons like switched-off or not reachable or network congestion, the message would not be discarded if it expires before getting delivered to User Mobile; it would be sent to the mail id registered by the user.
