Title of Invention

A METHOD OF ENCODING DATA IN ELECTRONIC FORM USING AN ENCODER FOR TRANSMISSION OVER A COMMUNICATION CHANNEL
Abstract An encoder (115) reads an ordered plurality of input symbols (110), comprising an input file (101) or input stream (105), and produces output subsymbol. The ordered plurality of input symbols are each selected from an input alphabet, and the generated output subsymbols comprise selections among an output subsymbols alphabet. The output subsymbol is generated using a function evaluator applied to subsymbols of the input symbols. The functions used to generate the output subsymbols can be XORâ€™s of some of the input subsymbols and these functions are obtained from a linear code defined over an extension field GF(2). In a decoder (155), output subsymbols received by the recipient are obtained from output subsymbols transmitted from one sender that generates those output symbols based on an encoding of an input sequence.
