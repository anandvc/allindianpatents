Title of Invention

"AN EXTENSION SCREED FOR A SCREED ASSEMBLY OF A PAVING VEHICLE "
Abstract An extension screed for a screed assembly of a paving vehicle, the screed assembly including a main screed having a screed plate, the extension screed comprising a base movably connectable with the main screed, a carriage movably connected with the base and configured to support an extension screed plate, wherein each one of the main and extension screed plates has a lower horizontal working surface, a horizontal actuator connected with the base and configured to linearly displace the carriage with respect to the base in opposing, generally horizontal directions; and a first vertical actuator connectable with the main screed and configured to linearly displace the base and the carriage with respect to the main screed in opposing, generally vertical directions so as to adjust the elevational position of the extension screed plate with respect to the main screed plate and a second vertical actuator is connected with the main screed and with the base, wherein the first and second vertical actuators are configured to linearly displace the base and carriage with respect to the main screed in opposing, generally vertical directions so as to adjust the elevational position of the extension screed plate with respect to the main screed plate and the first and second vertical actuators are configured to pivot the base and the carriage with respect to main screed so as to adjust a slope angle between the extension screed plate surface and the main screed plate surface. Full Text The present invention relates to an extension screed for a screed assembly of a paving vehicle.
Screed assemblies for paving vehicles are known and generally include a main screed having a frame connectable with a paver tractor and pne or more screed plates mounted to the bottom of tlie frame, Often, screed assemblies often include extension screeds mounted to the front or reai- of the main screed so as to be laterally movable with respect to the main screed, Such extension screeds provide the capability of forming a wider mat of paving material when the extensions are displaced laterally outwardly and to produce narrower mats when the extensions have been displaced inwardly toward the center of the main screed.

Referring to Figs. 3-6 and 13-15, as discussed above, the extension screed assembly 10 preferably includes two vertical actuators 16, specifically a first, inner actuator 17A connected with the inner base portion 20A and a second, outer actuator 17B connected with the outer base portion 20A. displaces the base 12.

In any case, the two preferred vertical actuators 17A, 17B preferably function to vertically displace and/or pivot the extension screed 10 relative to the main screed 3 in the following manner. Alternatively, the base 12 also pivots within the plane Py when one actuator movable portion 68 is held stationary while the other actuator movable portion 68 is displaced either upwardly or downwardly. In either case, pivotal displacement of the base 12 with respect to the main screed 3 adjusts a slope angle AS between the associated extension screed plate working surface 1 la and the proximal main screed plate working surface 4a.
