Title of Invention

' AN APPARATUS AND METHOD FOR ANNOTATING A DOCUMENT"
Abstract Apparatus and methods allowing users to review and add annotations (512,910) to a digital document (305,306). The document is presented to the user via an audio output device (310) and provides the user with the ability to annotate the document by speaking into an audio input device (310). The user may access the document from multiple locations using multiple types of devices.The user may, for example, begin reviewing a document in an automobile, continue to review the document via a cell phone, and finish reviewing the document from home using a regular phone line.
