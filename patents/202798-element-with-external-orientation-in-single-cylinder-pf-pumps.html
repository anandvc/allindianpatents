ELEMENT WITH EXTERNAL ORIENTATION IN SINGLE CYLINDER(PF)PUMPS
Abstract As per the invention herein described, in PF pumps, the element barrel fixing arrangement for the pump assembly has been dispensed with &. a simple methodology for barrel orientation has been evolved. As per the invention, the pump housing has been modified to access the barrel inlet port by an external tool. During the pump assembly, an external tool locates and orients the element barrel in the pump housing and there abr the Delivery Valve holder is tightened. The external tool is removed after tightening the Delivery Valve holder to the required torque.
Full Text In a PF type fuel injection pump, the element comprising of Plunger 6. Barrel assembly is the key component in pressuring the specified quantity of diesei fuel & delivering it in to the engine. The external size of the pump is mainly dependent on the size of the element and the maximum permissible peak pressure. If the size of the element barrel can be reduced then the overall size of the pump can be reduced.

The main object of the invention is to reduce the overall size of the element and in turn, the size of the PF pump, so that a compact PF pump can be offered for smaller engines.
In a conventional PF pump, the element barrel has twin stepped collar. The bigger step of the collar provides the sealing between pump housing and the element barrel, so that the diesel fuel is prevented from leaking downwards into the engine crank case. The smaller step of the barrel collar has a specially milled key way slot, which accommodates a locking / fixing pin protruding from the housing.

The pump housing has a small hole, into which a locating pin is pressed using interference fit This pin is made to protrude from the housing

ADVANTAGES OF INVENTION
This orientation of the barrel by an external tool reduces the number of components, simplifies the components and the assembly process. Besides, this concept helps in reducing the overall size of the pump without compromising on the sealing surface area.
