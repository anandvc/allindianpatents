Title of Invention

METHOD FOR PROVIDING AUTHORIZED SET OF QUALITY OF SERVICE (QOS) PARAMETERS WHEN A USER EQUIPMENT IS IN ROAMING
Abstract This invention provides a method and system foe enabling PCRF based functons when the user is in roaming. This invention relates to both normal case when the Qos is authorized successfully ans also to the error case where the QoS is rejected as it violates the local policy/User Subscription This invention relates to a system and method to enable the PCRF based QoS authorization in roaming architecture.
