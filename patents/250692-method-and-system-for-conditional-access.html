Title of Invention

"METHOD AND SYSTEM FOR CONDITIONAL ACCESS"
Abstract A method for granting access to a system (1) in which different services are distributed over a distribution network (4) and in which a user terminal (3) is used to decode these services, characterised in that access to services distributed over said distribution network is granted to said user terminal (3) by means of a virtual CA (Conditional Access) card, said virtual card including an identification of the user of said user terminal, which virtual card is downloaded over the distribution network (4) by means of an data stream (18) that conveys said virtual card from a system unit (17) to be received by said user terminal (3).
Full Text



We claim:
1. A method for granting access to a system (1) in which different services are distributed over a distribution network (4) and in which a user terminal (3) is used to decode these services, characterised in that access to services distributed over said distribution network is granted to said user terminal (3) by means of a virtual CA (Conditional Access) card, said virtual card including an identification of the user of said user terminal, which virtual card is downloaded over the distribution network (4) by means of a data stream (18) that conveys said virtual card from a system unit (17) to be received by said user terminal (3).
2. Method as claimed in claim 1, wherein the data stream (18) downloading the virtual card transfers a unique user identity of the user of said user terminal.
3. Method as claimed in claim 1 or 2, wherein the data stream (18) downloading the virtual card transfers functions as contained in a conventional CA card (16).
4. Method as claimed in any of the preceding claims, wherein the virtual card is stored in a memory (19) in the user terminal (3).
5. Method as claimed in claim 4, wherein said storage is effected in a flash memory.
6. Method as claimed in claim 4, wherein said storage is effected in a RAM memory.
7. Method as claimed in any of the preceding claims, wherein said download of the virtual card includes secured transfer.
8. Method as claimed in claim 7, characterized in that said secure transfer comprises encrypting.
9. Method as claimed in claim 7 or 8, wherein said secure transfer comprises having a user enter a code into the user terminal (3) before said virtual card may be received.
10. Method as claimed in any of the preceding claims, wherein said system unit is a VCD injector (17), the main task of which is to generate virtual cards.
11. Method as claimed in any of the preceding claims, wherein the user terminal (3) is a set-top-box (3).
12. Method as claimed in any of the preceding claims, wherein the distribution network (4) is a digital TV network.
13. Method as claimed in any of the preceding claims, wherein said services comprises at least one of the following: TV channels, TV programmes, movies or games.
14. Method as claimed in any of the preceding claims, wherein said downloading is ordered by a user by means of the user's mobile terminal, wherein a server device (20) connected to a mobile communication network sends an order to said server unit (17) to create a virtual card and send it to the user.
15. Method as claimed in any of the preceding claims, wherein said data stream includes an identity unique for said user terminal, by means of which said virtual card can only be downloaded to this user terminal.
16. A system distributing different services over a distribution network (4) and in which a user terminal (3) is used in order to decode these services characterized in that the system includes a virtual CA (Conditional Access) card for granting access to services distributed over said distribution network to said user terminal (3), said virtual card including an identification of the user of said user terminal, and means for downloading the virtual card over the distribution network (4) by means of a data stream (18) that conveys said virtual card from a system unit (17) for reception by said user terminal (3).
17. System as claimed in claim 16, wherein the system unit arranged to download the virtual card by means of the data stream (18) transfers a unique user identity of the user of said user terminal.
18. System as claimed in claim 16 or 17, wherein the system unit (17) arranged to download the virtual card by means of the data stream (18) transfers functions as contained in a conventional CA card (16).
19. System as claimed in any of claims 16-18, wherein the system includes a memory (19) in the user terminal (3) in which the virtual card is stored.
20. System as claimed in claim 19, wherein said memory is a flash memory.
21. System as claimed in claim 19, wherein said memory is a RAM memory.
22. System as claimed in any of claims 16-21, wherein said means for downloading of the virtual card includes means for secured transfer.
23. System as claimed in claim 22, wherein said means for secured transfer comprises encrypting apparatus.
24. System as claimed in an of claims 16-23, wherein said system unit is a VCD injector (17), the main task of which is to generate virtual CA cards.
25. System as claimed in any of claims 16-24, wherein the user terminal (3) is a set-top-box (3).
26. System as claimed in any of claims 16-25, wherein the distribution network (4) is a digital TV network.
27. System as claimed in any of claims 16-26, wherein said services comprise at least one of the following: TV channels, TV programmes, movies or games.
28. System as claimed in any of claims 16-27, wherein said user terminal has a unique identity and that said data stream includes this unique identity, by means of which said virtual card can only be downloaded to this user terminal.
