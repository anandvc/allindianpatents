As used in the claims of this patent specification, "service provider agent specifying information" shall mean all information which enables an access to a service provider agent and include an agent name, object class name and address name as well as a combination thereof.
As used in the claims of this patent specification, "service provider agent classification information" shall mean all information which specifies the class of servic¬es provided by a service provider agent, and is a concept including all information which identify an interpreter, a translator, a conversation check routine, a format check routine, an interpretation execution routine and an execution routine that are held by a service provider agent.
