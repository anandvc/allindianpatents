1. Technical Field
The present invention relates to mobile communication devices and, more particularly, to hybrid mobile stations capable of communicating in a data-only mode with a data network and in a voice mode with a voice network.

Summary of the Invention
An access network controller and a base station controller are formed to define an interface there between that enables the two systems to facilitate and respond to a voice call that, is to be terminated to a hybrid mobile station even though the hybrid mobile station is presently engaged or potentially is presently engaged in a data only call.
