Title of Invention

ELECTRONIC SCALES AND METHOD FOR THE OPERATION THEREOF
Abstract The invention relates to a set of electronic scales comprising an integrated computer comprising a display for presenting information to a user, an input unit for inputting selection data by a user, a memory for memorising a plurality of activatable profiles as sets of parameter values for allocating to corresponding sets of parameters, a data processing unit for controlling the display, the input unit and the memory and to determine and process measuring values in accordance with the actual activated profile, said measuring values being produced during processing of a weighing process performed by the user. In order to determine a profile, a user is requested, in a plurality of successive steps, to select one of several options presented by the display, the combination of the options presented during one step being dependent on the selection carried out in the preceding step. During the weighing process, the user is requested to perform in a plurality of successive steps an action requested by the display. The request is dependent on the type of actual activated profile and it only then takes place if the action requested in the preceding step is confirmed as carried out.
