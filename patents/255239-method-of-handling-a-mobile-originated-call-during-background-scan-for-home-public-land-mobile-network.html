Title of Invention

METHOD OF HANDLING A MOBILE ORIGINATED CALL DURING BACKGROUND SCAN FOR HOME PUBLIC LAND MOBILE NETWORK
Abstract The method of handling mobile voice call during background scan for HPLMN/HIGH priority PLMN comprising the steps of aborting the HPLMN/High priority PLMN Search whenever a user attempts to make call ;and proceeding with Mobile Originated call handling after the search abortion is completed. In the method an HPLMN search timer is re-started as soon as the search abortion is completed. In the method an HPLMN/High priority PLMN Search gets started/resumed as soon as the Mobile Originated call is finished.
Full Text FIELD OF INVENTION
The present invention in general relates to communication technology. Specifically the present invention relates to a method of handling the mobile originating calls while doing Background Scan for HPLMN /High priority PLMN.
DESCRIPTION OF THE RELATED ART
When the MS is in national roaming or international roaming, the MS always searches for HPLMN/High priority PLMN at periodic intervals as determined by periodic HPLMN search time written on SIM. If SIM doesn't have any value for the HPLMN search time, then a default of 60 minutes is considered. UE does periodic attempts for Background scan for HPLMN/High priority PLMN when the conditions for the search are met. If the MS supports more than one RAT, MS may search for the HPLMN/High Priority PLMN in all possible RATs the MS is capable of.
When the MS is in national roaming or international roaming, the MS always searches for HPLMN/High priority PLMN at periodic intervals. The existing method does not mention any thing specific to supporting Mobile originating calls while Background Scan for HPLMN/High priority PLMN is in progress. This is a cause of concern because user attempted calls during this time may not be successful or may be delayed, depending on the implementation of the UE. This problem could be more prominent if the periodicity of the Background scan is

high (Worst case is for every 6 minutes) and added upon to that if the MS is capable of operating in more than one RAT it will add extra time involved in doing Background scan for all the RATs it is capable of.
SUMMARY OF THE INVENTION
The primary objective of the present invention is to give the priority to Mobile originated calls during Background scan for HPLMN/High priority PLMN by aborting the HPLMN/High priority PLMN Search.
The solution is to abort the HPLMN/High priority PLMN Search whenever the user attempts to make call and proceed with Mobile Originated call handling after the search abortion is completed. The HPLMN search timer will be re-started as soon as the search abortion is completed or HPLMN/High priority PLMN Search could be started/resumed as soon as the Mobile Originated call is finished.
The invention is about handling of the Mobile originated calls while HPLMN/High priority Search is in progress by aborting the HPLMN/High priority PLMN Search.
Accordingly the invention explains a method of handling mobile voice call during background scan for HPLMN/HIGH priority PLMN comprising the steps of:
aborting the HPLMN/High priority PLMN Search whenever a user
attempts to make call ;and
proceeding with Mobile Originated call handling after the search abortion

is completed.
An HPLMN search timer is re-started as soon as the search abortion is completed.An HPLMN/High priority PLMN Search gets started/resumed as soon as the Mobile Originated call is finished.
