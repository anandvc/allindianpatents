Title of Invention

SEWING MACHINE WITH FUNCTION SWITCH
Abstract The invention relates to a sewing machine with function switch (12) with a plurality of operating elements (42a to 42f) for operation of switches (37a to 37f) which are connected by electric lines to a sewing machine control system (62); the switches (37a to 37f) that are allotted to the operating elements (42a to 42f) being arranged to call certain switching functions (fa to ff) of the control system (62); and the operating elements (42a to 42f) being disposed in an area of transition of a sewing machine head (5) to a sewing machine arm (4). The function switch (12) comprises an operating lever (54) which extends into an operator's working area (A) and which is mechanically coupled with an operating-lever switch (39; 39'); and in that the switch of the switches (37a to 37f) that is allocated to a selected operating element of the operating elements (42a to 42f) is electrically connectable to the operating-lever switch (39; 39') via a switching system (38a to 38f) such that the switching function of the switching functions (fa to ff) that is allocated to the selected operating element of the operating elements (42a to 42f) is switchable by the operating lever (54).
