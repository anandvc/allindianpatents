Title of Invention

DEVICE FOR PRODUCING SPECIAL YARN
Abstract A device for producing special yarn changes the speed of a front roller in accordance with a plurality of pattern data sets which show the change in thickness of spun yarn to produce special yarn corresponding to the pattern data including at least information on a pitch between slubs. A CPO controls a first servo-motor so as to spin yarn obtained through execution of a one cycle repetitive pattern in accordance with an executable program for executing the one cycle repetitive pattern including the pattern data sets greater than the pattern data sets stored in a work memory based on the pattern data sets stored in the work memory.
