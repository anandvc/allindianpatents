Title of Invention

LOCKER CONTROL SYSTEM USING MOBILE PHONE
Abstract Using Mobile phones has become most common in today's life style. Most of the locker systems are opened and closed using the keys and this methodology is insecure today that anyone can open the locker using a dummy key. To overcome this problem, we have designed a locker with IR Sensor, which detects the key when it is inserted and sends an electric pulse to the interfacing circuit. The circuit generates a digital signal which is sent to I the PC, which in turn is converted into an alert message viz., "Intruder detected on (date) at . (time) Enter your option J. 'YES' to open 2. 'NO' to close "to the corresponding user's I mobile phone. The user can send a reply as YES or NO. If 'YES' is the message, then the locker would be opened. If 'NO' comes, then the locker would be in its initial closed state and an alarm will be activated.
