Title of Invention

SDMA FOR WCDMA WITH INCREASED CAPACITY BY USE OF MULTIPLE SCRAMBLING CODES
Abstract Systems and methodologies are described that facilitate increasing system capacity in a code-limited WCDMA (e.g., TDD, FDD, â€¦) wireless communication environment. According to one aspect, a larger code space can be defined by introducing multiple code clusters within a sector, wherein each cluster has a unique scrambling code. Codes within a cluster can have orthogonal Walsh sequences that can be assigned to user devices to facilitate communicating over a wireless network and can overlap with codes in another cluster. The unique scrambling code assigned to each cluster can ensure that duplicate Walsh sequences in another cluster in the same sector appear as a pseudo-noise codes.
Full Text The following description relates generally to wireless communication, and, amongst other things, to system capacity in a WCDMA wireless communication environment.
