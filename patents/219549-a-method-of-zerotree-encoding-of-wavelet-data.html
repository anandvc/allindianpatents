Title of Invention

A METHOD OF ZEROTREE ENCODING OF WAVELET DATA.
Abstract A computer system (100) includes a memory (118) and a processor (112). The memory (118) stores a program (119) to cause the processor (112) to provide wavelet coefficients that indicate an image. The processor (112) represents each wavelet coefficient as a collection of ordered bits, and the processor (112) codes the bits of each order to indicate zerotree roots that are associated with the order. Thus, unlike conventional zerotree coding thresholds are not computed in subordinate pass to identify insignificant values, but bits are classified: if bit equals to "1" it is a Positive node (P), if bit equals to "-1" it is a Negative node (N), a zerotree root is coded as R and an Isolated zero is noted IZ.
