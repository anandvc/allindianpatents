Title of Invention

METHOD OF ASCERTAINING CONTROL PARAMETERS FOR A CONTROL SYSTEM
Abstract A method of ascertaining control parameters for a control system having a controller (14) and a plant (10) includes establishing a model (70) of the plant and a model (64) of the controller and calculating a performance index for a closed-loop system as a function of controller parameters, accounting for selected stability margins.
