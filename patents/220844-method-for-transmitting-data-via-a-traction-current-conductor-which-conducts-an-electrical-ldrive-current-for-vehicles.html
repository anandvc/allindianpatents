Title of Invention

METHOD FOR TRANSMITTING DATA VIA A TRACTION CURRENT CONDUCTOR WHICH CONDUCTS AN ELECTRICAL LDRIVE CURRENT FOR VEHICLES
Abstract The invention is based on the object of disclosing a method for transmitting data via a traction current conductor which simultaneously conducts drive current for a vehicle and in which an interruption in the data transmission by an arc occurring between a vehicle-end drive current collector and the traction current conductor is reliably avoided. The object is achieved by means of a method for transmitting data (Q) via a traction current conductor (15), in which at the transmitter end at least two data signals containing in each case the data (Q) in an identical form are fed into the traction current conductor (15) with different frequency channels, In the event of an arc occurring at the receive end, the received data of that data signal or those data signals whose amplitude does not drop below a ration necessary for achieving a specific bit error probability with respect to the amplitude of the interference signal S(f), is selected for evaluation.
