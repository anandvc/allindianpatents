Title of Invention

METHOD OF PROVIDING RIGHTS DATA OBJECTS
Abstract A method of providing rights data objects for issuing to a device having access to encrypted content belonging to one of a plurality of events and provided with an indication of a location from which the rights data object may be requested and 5 event information uniquely associated with the event, which device includes an agent function for providing a request to a device (24) issuing rights data objects from the indicated location and data representative of the event information,includes: receiving the request and data representative of the event information, generating a rights data object, including event key event information enabling the content data belonging to the event uniquely associated with the information to be decrypted, and is characterised by generating the event key information using a cryptographic function that operates on at least a part of the event information.
