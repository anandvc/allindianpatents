Title of Invention

HOME AUTOMATION AND SECURITY SYSTEM USING GPRS MOBILE PHONE
Abstract Home Automation and Security system using GPRS mobile phone where in all the home appliances are connected to the parallel port of the home computer through the interfacing switching circuits, these circuits will be provided to user on registration with unique suerid and password, GUI (graphical user interface) is designed using VS.NET which is hosted on system provider's webserver and can be accessed on GPRS mobile phone, through which user can control as weel as monitor all his home appliances, this home computer is connected to internet; Infra-red sensors (IR transmitter and receiver), heat sensors are provided and used for security reasons which are connected to same parallel port, on receiving signals from thses security circuits, home computer will send SMS, FAX, alert message on YAHOO, MSN, AIM, ICQ messengers through internet; this will be sent using software designed in VS.NET residing on home computer; the communication between home computer and user's GPRS mobile phone takes place through webserver.

Technical Field:
This invention relates to a Home automation and security system using GPRS mobile phones.
Prior Art:
Existing home automation systems work only on systems having unique global IP address which is the rare case in general internet users; most of the users take internet connection through either phone line or proxy server and don"t get unique IP address; so to use existing systems on such computers it need to be converted to web server.
All the existing systems use either XI0 or Blue tooth devices, which can operate only for some distance.
Existing systems use very costly and complicated interfacing circuits (e.g. X10, Blue tooth etc devices) for the connection between PC ports. Existing systems cannot control speed of fan/motor efficiently. Existing systems do not provide proper scheduling of the appliances. Hence it is necessary to provide a home automation and security system to obviate all the problems/drawbacks pointed out above.
Objects of the Invention
The main object of this invention is to provide a home automation and security system using GPRS mobile phones system working on all kinds of connections.
Further object of this invention is to provide a home automation and security system using GPRS mobile phones system operated from any part of the world.
Further object of this invention is to provide a home automation and security system using GPRS mobile phones system by using very cheap and simple interfacing circuits, which makes it cost effective.

Further object of this invention is to provide a home automation and security system using GPRS mobile phones system with variable speed control means.
Further object of this invention is to provide a home automation and security system using GPRS mobile phones system with system for scheduling the appliances.

Brief description of Invention
Home Automation and Security system using Web enabled Mobile Phone is an entirely customisable, user friendly and flexible system wherein the user can control various home appliances like lights, TV, refrigerator, fans etc including the speed of the fan, using web enabled mobile phone, any computer with internet connection or connected to LAN. The user can control any of his appliances from his mobile phone. He can switch them
4

ON and OFF whenever he requires. This means that you will be able to control your house from any corner of the world.
The system is deployed on a web server so that it can be accessed through Internet. The database is accessed and updated through the system. The Server Database uses Microsoft Access. It has two tables; one table stores the username, password, and the list of appliances and there current status (ON or OFF), the second table is for scheduling of the home appliance. The user registers on the website and provided with a unique user id and password. The user gives the list of devices in his house, which needs to be accessed by him constantly.
The user access the website through his mobile phone. He then logs into the site using the username and password provided to him. He is then provided with a list of his home appliances and an option for making it ON or OFF and scheduling the device. He can do the required changes here. He can even monitor and change the status of the device, like; the user can change the speed of the fan through this. He can change the speed from the current speed to the required speed directly instead of going through steps.
In case of fire (Heat Sensor) and burglar (IR Sensors) at home, the home owner will get the SMS, FAX, alert message on YAHOO, MSN, AIM, ICQ messenger instantly (using XML Web services).
The system is also provided with a software where the user can control the home appliances using his voice from home using said software and voice recognition system doesn"t need to train the computer and it will recognize any voice hence its completely user friendly.

As shown in figure 1 the web control system is deployed on a web server so that it can be accessed through Internet. The database is accessed and updated through the system. Refrigerator, fan etc. in his house, which needs to be accessed by him constantly.
On the home PC, which the user uses to control the appliances, one page runs continuously; this page is the page of website accessed through the Internet. This page reads the status of the device (whether ON or OFF) from server database and sends a query string to the local host page, which controls the computer. This page is refreshed continuously on every 5 seconds. This page uses port programming. When there is a request to perform some status change to the page then it sends a signal through the parallel port and the value of the register is input to the parallel port.
The client computer must be installed with Microsoft .NET Framework. It should also have some web browser like Internet Explorer, Netscape, Mozzilla etc.
The user access the website through his mobile phone. He then log into the site using the username and password provided to him. He is then provided with a list of his home appliances and an option for making it ON or OFF and scheduling the device. He can do the required changes there. He can even monitor and change the status of the device, like; the user can change the speed of the fan through this. He can change the current speed to the required speed directly instead of going through steps.
A home page will be displayed after getting logged in from site. This home page consists of a combo box having list of all home appliances and other combo box having ON/OFF option. A button is provided to apply the changes. A hyperlink is provided to scheduling and Fan/motor control pages.
The various Features of the Home automation and security system are discussed
in detail
i) Scheduling Feature :
The user can also schedule the system for usage. In scheduling the user sets two times between which he wants the appliance to be ON. An example for scheduling would be the task of switching the water pump ON and OFF between two times everyday. The user can schedule the system such that the water pump switches ON at the time set by the user and switches OFF at the time set by the user automatically. Hence there will not be any problem of forgetting to switch it ON or OFF everyday.
6

ii) Security Features:
The security system provided by this system sends an SMS (Short Messaging Service), FAX and instant alert message on YAHOO, MSN, ICQ messenger to the house owner whenever the security of his house is breached. The SMS will be sent in 0.32 seconds, the FAX will be sent in 0.26 seconds and instant alert message on YAHOO, MSN, ICQ, and AIM messenger will be sent in 0.28 seconds. The SMS or FAX can be sent to as many people as desired.
The system detects the break of path of Infra Red rays of transmitter receiver which will be placed at suitable places at home and system send the SMS, FAX, alert message on YAHOO, MSN, ICQ, AIM messengers which is sent without using any mobile phone or FAX machine.
a) Fire Protection:
Heat sensors have also been used which will detect fire in the house and send an SMS, FAX, alert message on YAHOO, MSN, ICQ, AIM messengers to the nearby fire station giving the address of the house and the time the fire was detected. An SMS and FAX will also be sent to the house owner regarding the accident. All these three (SMS, FAX, alert message) features have been kept just in a case one of them is not working the others will work.
b) Burglar Security Using Infrared Waves:
An Infrared transmitter and receiver is used. The system (receiver plus transmitter) is embedded in near doors or windows or anywhere in home. Whenever there is an intruder the circuit is broken. The receiver sends a signal to the computer and the computer will send SMS, FAX, alert message on YAHOO, MSN, AIM, ICQ messengers; this will be sent using software designed in VS.NET.
iii) Poor bell Response:
If you want to leave any message for the visitors at the home in your absence then you just have to type the message in textbox of our software. This message will be played when the visitor presses the doorbell.
7

V) Voice Automation:
A software is provided to user where he can control home appliances using his voice from home .The voice recognition system doesn"t need to train the computer and it will recognize any voice so its completely user friendly.
There is provided a reduced the dictionary of the voice recognition machine so that it doesn"t compare your voice with million of words But it will compare voice with only predefined words from dictionary so thereby eliminating the wrong detection.
All interfacing circuits of system are cost effective hence making the system most cost effective among all other existing systems.
Method of controlling the home appliances from any part of the world using GPRS mobile phone as referred in fig 2 comprises the following steps
a) Selecting the particular home appliance on the GPRS mobile phone for performing the desired operation such ON/OFF, Speed control etc,
b) Sending a signal to the web server by a GPRS mobile phone for performing the said desired operation, such that this signal will change the data stored in the web server,
c) Home PC which is continuously checking the web server identifies the change of data in a web server,
d) Applying the changes to the particular appliance by generating the signal through parallel port of a home PC,
e) On receiving above said signal the interfacing circuit actually
performs the above said operation on appliance.

security module including infrared sensor and heat sensors to protect home from fire and theft; said sensors are connected to the parallel port of the user computer, on receiving the signals from said security module user computer send the SMS, FAX or alert message through internet;
