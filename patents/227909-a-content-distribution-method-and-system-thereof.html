Title of Invention

A CONTENT DISTRIBUTION METHOD AND SYSTEM THEREOF
Abstract In a content supply system (100), a user PC (106) is used to make a reservation for a time zone by accessing a server use reservation management center (101) so as to use a streaming server (102) to perform live distribution. When the user PC (106) transmits reservation request information to the server use reservation management center (101) through the Internet (103), a reservation ID used only for authentication about the reservation is created if the reservation is accepted and transmitted to the user PC (106) from the server use reservation management center (101) through the Internet (103). If the user PC (106) sends a distribution request to the streaming server (102), authentication using the reservation ID is performed.
