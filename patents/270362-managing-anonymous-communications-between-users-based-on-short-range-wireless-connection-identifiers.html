Title of Invention

MANAGING ANONYMOUS COMMUNICATIONS BETWEEN USERS BASED ON SHORT-RANGE WIRELESS CONNECTION IDENTIFIERS
Abstract Method for managing in a network equipment (1) of a communication network (100) anonymous communications between a first and a second mobile user apparatus (120), the method comprising a) receiving from the first mobile user apparatus (120), through a first network connection, data indicating that a short-range wireless connection between the first mobile user apparatus (120) and the second mobile user apparatus (120) has been established, said data comprising a second unique short-range wireless connection identifier associated with the second mobile user apparatus (120) and a message for the second mobile user apparatus (120); b) by using said second unique short-range wireless connection identifier, obtaining a network address associated with the second mobile user apparatus (120) ; and c) by using the network address obtained in b), sending data to the second mobile user apparatus (120) through a second network connection, said data comprising the message received in a) from the first mobile user apparatus (120) and a first anonymous identifying code univocally associated with the first mobile user apparatus (120).
