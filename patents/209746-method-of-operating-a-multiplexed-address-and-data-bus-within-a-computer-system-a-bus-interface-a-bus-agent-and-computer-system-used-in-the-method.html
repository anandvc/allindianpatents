Title of Invention

METHOD OF OPERATING A MULTIPLEXED ADDRESS AND DATA BUS WITHIN A COMPUTER SYSTEM, A BUS INTERFACE, A BUS AGENT AND COMPUTER SYSTEM USED IN THE METHOD
Abstract An apparatus, for operating a multiplexed 128-bit external bus within a computer system, includes arbitration logic that arbitrates between contending address and data requests according to the number of outstanding data and/or snoop transactions on the external bus. In response to the outcome of this arbitration, selection logic grants the external bus to either the contending address or data request. The arbitration logic may compare the number of outstanding data transactions to a predetermined data threshold number when performing the arbitration, this data threshold number being dynamically alterable by an application program or operating system so as to optimize external bus throughput under predetermined conditions.
