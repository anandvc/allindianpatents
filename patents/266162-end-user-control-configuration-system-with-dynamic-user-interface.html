Title of Invention

“END USER CONTROL CONFIGURATION SYSTEM WITH DYNAMIC USER INTERFACE”
Abstract An end user control configuration (EUC) system with dynamic user interface provides a user friendly tool for consumers to self-provision and manage network services such as voice over internet protocol (VoIP) Services................
Full Text This disclosure concerns a dynamically updating graphical user interface for a manegment system for provisioning and managing voice over internet protocol (VoIP) Services
