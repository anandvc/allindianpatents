Title of Invention

"AN APPARATUS AND A METHOD THEREOF FOR DEMODULATING A CODE WORD"
Abstract An apparatus and method thereor for dimodulating a code word The demodulation apparatus includes a code table including a plurality of the code words, wherein similar ones of the code words are arranged to be grouped together; and a soft demodulator to calculate probabilities of individual bits that constitute the code words, and to generate a soft demodulation value of the data word.

We Claim:
1. An apparatus and a method thereof for demodulating a code word having a
predetermined bit length greater than a first predetermined bit length, the code
word resulting from modulation of a data word having the first predetermined bit
length, the apparatus comprising:
a memory device incorporating a code table, the said code table comprising a
plurality of the code words,
a code word grouping means coupled to the memory device for grouping together
similar ones of code words and storing said similar ones of the code words
together on the memory device, wherein the code table is formed by converting
one column, containing all code words, into a predetermined number of columns;
and
a soft demodulator linked with the said memory device for calculating
probabilities of individual bits that constitute the code words, and for generating a
soft demodulation value of the data word.
2. The apparatus as claimed in claim 1, wherein the code words are allocated to the predetermined number of columns such that common portions of the code words are arranged in common ones of the predetermined number of columns.
3. The apparatus as claimed in claim 2, wherein common neighboring portions are written one time in a single section of the respective common ones of the predetermined number of columns.
4. The apparatus as claimed in claim 1, wherein the input signal is a run-length limited code signal transmitted through channel or output from a soft channel detector.
5. An apparatus for demodulating a code word substantially as herein described with reference to the accompanying drawings.
