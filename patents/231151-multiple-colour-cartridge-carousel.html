Title of Invention

MULTIPLE COLOUR CARTRIDGE CAROUSEL
Abstract The Multiple Colour Cartridge Carousel provides the ability to house dedicated colour cartridges for multiple (typically 5) colours in one single carousel, and to change colour by merely indexing the carousel. Thus, the user requires only one carousel for multiple colours. The solution also replaces six cartridges place side-by-side with a pair of cartridges placed on below the other. Multiple (typically 5) such pairs are mounted in separate compartments of the carousel. In addition, these cartridges get alignead under one single pulse valve. This one pulse valve, being common to all the sets of cartridges, performs the function of the multiple pulse valves in traditional cartridge booths.
Full Text

Prior Art:
Powder spray booths with cartridge recovery systems offer some major benefits. By dedicating a cartridge module for each recovered colour, they provide a high degree of colour integrity during colour change. In addition, they provide for relatively quick colour change, a means of spraying powder paint with virtually no waste and a clean and safe work environment. Yet, a traditional cartridge booth is an expensive solution, because it involves multiple numbers of the same pieces of hardware. Among other things, multiple numbers of the following items are required:
1. the cartridge module
2. and the pulse valves.
For every colour that is to be reclaimed, the user needs to invest in one cartridge module. In other words, to be able to recover five colours, one requires five cartridge modules.
Further, each cartridge module has multiple (typically six or more) cartridges, and such cartridge needs to be cleaned periodically by a reverse pulse of high pressure compressed air, controlled by a pulse valve. A module containing six cartridges requires six pulse valves.

This requirement of multiple pieces of hardware is compounded by the requirement of additional floor space for storing idle cartridge modules. Where floor space is expensive, this can significantly add to the cost of a powder coating system.
The cost of such an installation could be considerable decreased if these multiple requirements could be done away with.
Statement of invention
The solution provides the ability to house dedicated colour cartridges for multiple (typically 5) colours in one single carousel, and to change colour by merely indexing the carousel. Thus, the user requires only one carousel for multiple colours.
The solution also replaces six( or more) cartridges placed side-by-side with a pair of cartridges placed one below the other. Multiple (typically 5) such pairs are mounted in separate compartments of the carousel. In addition, these cartridges get aligned under one single pulse valve. This one pulse valve, being common to all the sets of cartridges, performs the function of the multiple pulse valves in traditional cartridge booths.

The invention will now be described in greater detail with reference to the accompanying drawings wherein the Fig.l shows Nordson Quintet Powder Booth Fig.2 shows Nordson Quintet Five-Segment Carousel.
Description of the Invention
1. Much of the high cost of a traditional powder booth is due to the multiple numbers of the same hardware. To handle multiple colours, such a booth requires one cartridge module for each recovered colour. Each cartridge module is an expensive piece of equipment that also occupies expensive floor space in addition, the booth requires as many pulse valves as there are cartridges in each module. So, for a typical six-cartridge module, the booth would require six pulse valves.
In the concept being disclosed, multiple colours can be recovered- using one multi segment carousel (Fig.2). The Nordson Quintet Powder Booth which embodies this concept uses a carousel containing five air-tight segments. One pair of cartridges is mounted inside each of these segments, giving the user up to five pairs of cartridges completely separated from each other and mounted on one single carousel. Thus, with one carousel, the user can spray up to five different colours.

2. By using larger cartridges and by mounting them one below the other, the number of pulse valves is also reduced from six in a traditional cartridge booth to only one in the Nordson Quintet Powder Booth.
3. The carousel is placed on five peripheral castor wheels and one central castor wheel with an axis that is coaxial with that of the carousel. The central wheel is guided by a horizontal wheel that travels within a pair of rails grouted into the floor.
4. In order to change colour, the booth and other equipment is first cleaned out as usual. Then the carousel is withdrawn in a linear motion. This is aided by the horizontal wheel which is constrained to move within the guide rails. Next the carousel is indexed so as to bring the required segment in front of the booth opening. The carousel is pushed back in position with the cartridges of the required colour now exposed to the inside of the booth.
When locked in position, the cartridges of the required colour are placed exactly below one single pulse valve housed in the fan housing above.

5. The concept offers a unique advantage to small users who do not need to spray five colours initially. Such customers need to populate only as many segments of the carousel as they need to spray colours. So, a user who initially needs to spay only two colours needs to populate only two segments with cartridges. When more colours are needed, all the user needs to do is to buy the additional sets of cartridges and populate the remaining empty segments. This allows the user to postpone his expenditure on cartridges to the point when they are actually required.
The advantages of this unit are as follows:
1. By eliminating multiple numbers of colour modules and pulse valves, the Nordson Quintet Powder Booth offers a low cost solution.
2. The solution provides ability to recover powders of multiple colours with high degree of recovery efficiency.
3. By eliminating the need for multiple colour modules, the solution uses lesser floor space.

We claim:
1 That a Multiple colour cartridge carousel housing dedicated color cartridges for multiple ( typically 5) colours will serve in recovering the multiple colors being sprayed.
2 That in a Multiple colour cartridge carousel, where two large cartridges per colour are used instead of six will allow the use of one pulse valve to clean the cartridges instead of six.
3 That a colour change in a multi Colour Cartridge carousel is effected as follows:
The carousel, aided by, horizontal wheels guided in rails is withdrawn in a straight line away from the booth. The carousel is then indexed so as to bring the required segment in front of the booth opening. The complete carousel is then pushed back and again coupled to the booth.
