Title of Invention

A SYSTEM FOR AUTOMATICALLY TRACKING PLURALITY OF MOBILE ASSETS
Abstract The present invention provides an economic system for automatically tracking mobile assets over long distances using global system for mobile (GSM) communication network. The system according to the invention is capable of tracking the mobile assets automatically and if required provide instructions to the driver of the vehicle carrying the mobile assets. The mobile assets are provided with proprietary access devices which are in constant contact with communication towers of GSM communication network. The location based information of the mobile assets will be the location based information of the nearest GSM communication tower and this information is transferred through the GSM communication network to a central command center. The central command center is equipped with a computer installed with a custom made software. The central command center updates the data base, provide instructions to the unit located with the mobile assets and pass the information to corporate web servers and to any other desired terminals based on security and password protocols.
