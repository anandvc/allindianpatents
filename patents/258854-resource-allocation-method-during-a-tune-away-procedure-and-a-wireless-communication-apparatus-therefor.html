Title of Invention

RESOURCE ALLOCATION METHOD DURING A TUNE AWAY PROCEDURE AND A WIRELESS COMMUNICATION APPARATUS THEREFOR
Abstract A resource assignment method during a tune-away procedure comprises determining a first instance in time that a tune-away will initiate with respect to an access terminal, determining a second instance in time that corresponds to a latest frame boundary associated with the access terminal prior to the instance in time that the tune-away initiates, and determining resource assignment parameters associated with the access terminal and implementing the resource assignment parameters at the second instance in time. The method can further comprise ceasing communications to the access terminal over a forward link at the second instance in time and discontinuing monitoring reverse link control channels associated with the access terminal at the second instance in time.
