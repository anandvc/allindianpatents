Title of Invention

METHOD FOR SORTING AND DISPLAYING MENU OPTIONS IN A MULTI FUNCTIONAL PERIPHERAL BASED ON FREQUENCY OF USAGE OF THE MENU OPTIONS
Abstract The invention describes a method fro sorting menu options in a Multi-functional Peripheral (MFP) based on the frequency of usage of the menus so as to reach the most used menu option by least number of key presses. The MFP maintains a usage history for each menu (or sub-menu) option selected by the user which gets incremented every time a user selects a particular menu option. This can be implemented for global usage of the menu and also at a user level.
