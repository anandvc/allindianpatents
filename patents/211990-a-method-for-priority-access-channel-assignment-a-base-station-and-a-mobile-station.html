Title of Invention

A METHOD FOR PRIORITY ACCESS CHANNEL ASSIGNMENT, A BASE STATION AND A MOBILE STATION
Abstract A priority access channel assignment system and a messaging protocol for wireless mobile stations. A mobile station (MS1, ..., MSIO) originates a call that manually or automatically includes a request for priority access. If system resources are unavailable to process the call immediately, the origination message is processed by a base station (BS1, BS2) and a mobile switching center (MSC) using a plurality of messaging protocols to determine the time at which the call was originated and the priority associated with the call. Based on priority and arrival time, the base station (BS1, BS2) queues the call where higher priority calls are placed in a higher position in the queue than lower priority calls. Calls having identical priority may be placed in the queue in order of the time of arrival. When system resources become available, the pending call with the highest priority is assigned a channel and the call is processed in a conventional manner.

BACKGROUND OF THE INVENTION
Cellular telephone technology has evolved from relatively simple voice communication with limited geographical areas of coverage- to sophisticated communication technology for voice communication, voice messaging, and data messaging virtually anywhere in the world. The number of mobile stations (i.e., users) has grown significantly in recent years. For example, in the United States, a large percentage of the population owns a cellular telephone. The implementation of complex communication systems, such as described above, require communication standards and protocol^ that allow different organizations and manufacturers to maintain compatibility of the overall system. For example, many standards for cellular telephone communication have been established by the Telecommunicatiolns Industry Association (TIA) and the Electronic Industries Association (E1A). j There are a number of TIA/EIA standards for cellular telephone communication. For the successful completion of a cellular telephone call, the cellular telephone system must have sufficient resources available to process the call request, j For example, a call originated by a mobile station requires that a channel be available to assign to that subscriber. During periods of peak communication traffic, a large number of users may utilize all available resources. Under these circumstances, a mobile station may find that the cellular telephone system cannot process a call. The user is forced to try again at a later time when traffic volume has diminished. . ^ Howeyer, the present invention allows prioritization of phone"calls ancl assigns channels based on the assigned priorities. Other subscribers may pay a premium; for high priority access to the cellular system. The PACA system and protocol of the present invention allows a mobile station to obtain priority access to a communications channel by gueuing call ■ origination requests when channels are not currently available. The system provides for multiple priority levels and prioritizes calls based on the priority level such that higher priority call requests are queued ahead of lower priority requests. Thus, a higher priority call may be queued ahead of a lower priority call, even though the lower priority call has been in the queue longer. Call requests having the same priority are queued in the order of time of arrival.
The system provides the user with an indication of their queue position (e.g., 8th in line) and alerts the mobile station when a channel is available. In addition,!the PACA system and protocol of the present invention allows the subscribejr to roam to another cell with a pending PACA request. If the new cell in which the subscriber is located has an available channel, the call request will be serviced immediately. IH the subscriber roams to a cell in the current serving system in which resources are not presently available, the PACA request i^ transferred in a manner that preserves the subscriber's place in the queue. Finally, the PACA system and protocol of the present invention cancels

PACA requests under certain conditions. The protocol for each of these above-described features will be described in greater detail herein.
