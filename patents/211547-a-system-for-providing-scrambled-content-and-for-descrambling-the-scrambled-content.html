Title of Invention

A SYSTEM FOR PROVIDING SCRAMBLED CONTENT AND FOR DESCRAMBLING THE SCRAMBLED CONTENT
Abstract A system for providing scrambled content, comprising a control word generator, each control word including a control word identifier, a scrambler for providing a stream of scrambled data packets, and an encryption device for providing entitlement control messages (ECM's). One or more consecutive packets are scrambled using the same control word (CW). Each packet includes a control word identifier identifying the control word used. The ECM encryption device provides ECM’s including previous control word (CW<sub>p</sub>), a current control word (CW<sub>c</sub>) and a next control word (CW<sub>n</sub>). A system for descrambling scrambled content comprises a descrambler for descrambling the scrambled content and a decryption device for devrypting ECM’s to obtain control words to the descrambler and the descrambler descrambles the data packets of the scrambled content using a control word having a control word identifier corresponding with the control word identifier of the data packet to be descrambled.
