Title of Invention

AUTOMATIC TEMPERATURE CONTROLLING AND RECORDING SYSTEM FOR PRE/POST HEATING APPLICATIONS
Abstract The present invention 'Automatic Temperature controlling and recording system for pre/post heating applications' comprising:- An infra-red sensor(7)for sensing temperature which is fixed on a rigid base(6)at a distance of approximately 1000 mm from the object under test. A processor(1)which converts the data received from sensor into voltage. A industrial laser source which determines the position of infra-red sensor(7). A PID temperature controller(2)which is used for maintaining the temperature within a specific range. Solenoid valves that work in coordination with PID controller. A temperature recording mechanism(3),which consists of a pressure sensitive paper.

Technical Details;-
The present invention "Automatic Temperature cantrolling and recording system for pre / post heating applications" is designed with a goal of providing accurate heating of weld Joints to Improve held quality. The new Invention is designed in such a way, so as to have higher speed and hence increased productivity it eliminates all the drawbacks of existing systems and fulfills all the desired objectives. The present invention "Automatic Temperature controller and recording system for pre / post heating applications" will now be explained in details with the help of accompanying diagrams.
Figure 1 shows the front panel of the system Figure 2 shows flow diagram of the process.
Figure 1(a) Illustrates Infra-red sensor prccessor 1, PID controller 2, Temperature recorder 3, Gas headers - ON/OFF 4, Gas headers 5.
Figure 1(b) Illustrates rigid base 6, Infra Red Sensor 7. Figure 2 illustrates steps 1 to 6 of the flow diagram
The system consist5 of an Infra-red sensor (7) for sensing the temperature. As illustrated in step 1 of figures 2, this IR sensor can accurately sense temperature of an object of which Is 1 meter away from It. A fine laser beam Is used to decide the fixing of the IR sensor.
The sensor Is mounted on a rigid base (6) at distance of approximately 1000mm from the object under test. As Illustrated In step 2 of Figure 2, the tenperature sensed by the IR sensor is given to a prccessor (1). The processor-converts the data into appropriate voltages. These voltages are then passed to a temperature controller (2) for controlling purpose (step 3).

The temperature controller, which is a RD controller (2), maintains these voltages within a specific predetermined range. The range can be any values between 0° C to 500° C. it avoids any under voltage or over voltage conditions (step 4).
The output of the RD controller is given to solenoid valves, which then operate to control the temperature (step 5). The same output Is given to a temperature reorder (3). The recording system Is a pressure sensitive paper., which can record a temperature from 0° C to 600° C (step 6), The system can record and control the temperature within a range of 0° C to 500o C with an accuracy of + 1 % of reading.
