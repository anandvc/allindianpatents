Title of Invention

AN IN-VITRO PROCESS FOR THE DETECTION OF THE ACTIVITY OF GAMMA-SECRETASE
Abstract This invention relates to an in-vitro process for the detection of the activity of y-secretase wherein A. a transgene is used which encodes a fusion protein comprising: a) a first nucleotide sequence which codes for a protein which contains the amino acid sequence GAIIGLMVGGVIA TVIVITL VML (SEQ ill. NO.1) wherein said protein can be cleaved within said amino acid sequence by y-secretase present in a cell, whereby a first partial protein, which contains the amino acid sequence GAIIGLMVGGVV (SEQ ill NO.2), and a second partial protein, which contains the amino acid sequence VIVITLVML (SEQ ill NO.3) are formed, b) at the 5' end of the first nucleotide sequence, ,a second nucleotide sequence which codes for a signal peptide, c) a promoter and, d) further coding nucleotide sequences which code for a protein which is expressed as a fusion protein with the first partial protein and the second partial protein and which is used for the detection of the second partial protein, and, if appropriate, further non-coding nucleotide sequences; B. this transgene is incorporated into a human cell or a non-human eukaryotic cell and the fusion protein is expressed; c. the fusion protein is cleaved within the amino acid sequence SEQ ill NO.1 by y-secretase present in the cell, whereby a first partial protein and the second partial protein are formed and D. the second partial protein is detected, characterized in that with the exception of SEQ ill No.1 said fusion protein does not contain one or more peptide motifs acting as a signal for endo- or exocytosis and/or protease cleavage site.
