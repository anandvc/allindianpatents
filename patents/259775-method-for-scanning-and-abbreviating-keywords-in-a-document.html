Title of Invention

METHOD FOR SCANNING AND ABBREVIATING KEYWORDS IN A DOCUMENT
Abstract A method for Optical Character Recognition (OCR) in the scanning process wherein pre-stored symbols or keywords are replaced with the stored words or abbreviations located in the memory according to the MFP function, is disclosed. This invention explains a method to scan and performing the abbreviation in an MFP comprising the steps of: adding customized keywords and abbreviations through scanning; forming Groups of scanned keyword through standard UI; mapping groups to Functions, through standard UI; and performing the abbreviation in scanned output if keyword is present in scanned data, wherein if the keyword is not present in the scanned data the process is ended without abbreviation.

SUMMARY OF THE INVENTION
The invention describes a method and a system for performing operations like scanning a shorthand script and conversion of the shorthand script to an expanded script for further editing. The method enables a user to customize a group of keywords generally used by the user. Presently, a system such as an optical card recognizer (OCR) or a scanner is required to scan the shorthand script. Further, a stenography system is required to convert the shorthand script to the expanded script. In the present invention, the user can perform the scanning of the shorthand script and conversion of the shorthand script to the expanded script using a mUItifunctional peripheral (MFP). The user can further edit the expanded script as

Adding customized keywords and abbreviation through scanning involves printing Keyword Defining Sheet from MFP through standard UI. Keyword Defining Sheet is a pre-stored sheet in MFP where keyword and abbreviation is entered. Adding customized keywords and abbreviation through scanning involves entering the Keyword and abbreviation and scanning the said sheet and storing the keyword list in MFP. Forming Groups of scanned keyword through standard UI involves entering the keyword group name. Forming Groups of scanned keyword through standard UI involves selecting Keyword from Keyword List and adding to the corresponding Group wherein each group is assigned to the various operations provided by the MFP. Mapping groups to Functions, through standard UI involves selecting a function for each group. The said function is Scan2email/fax/netsacn. A Keyword
