Title of Invention

A METHOD FOR A HOST COMPUTER TO ESTABLISH A COUPLING BETWEEN A LOGICAL STORAGE AREA OF A STORAGE DEVICE AND A VIRTUAL STORAGE AREA OF THE STORAGE DEVICE.
Abstract A host computer to establishes a coupling between a logical storage area (82) of a storage device (30) and a virtual storage area (84) of the storage device (30) by sending a first command (404) to the storage device to register the logical storage area. Following the first command, the host computer sends a second command (414) to the storage device to relate the logical storage area to the virtual storage area. Following the second command, the host computer sends a third command (424) to the storage device to activate the coupling between the logical storage area and the virtual storage area, where following activating the coupling, pointers of the virtual storage area point to sections of the logical storage area and where a write to the logical storage area causes data to be copied (358) from the logical storage area (82) to an other area of the storage device (30).
