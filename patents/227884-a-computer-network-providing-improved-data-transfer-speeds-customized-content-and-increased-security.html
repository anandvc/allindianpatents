Title of Invention

A COMPUTER NETWORK PROVIDING IMPROVED DATA TRANSFER SPEEDS, CUSTOMIZED CONTENT AND INCREASED SECURITY
Abstract This invention relates to a device for providing improved data transfer speeds and customized content in a configuration involving multiple content servers capable of providing the same content in which at least a first set of content servers or load Directors is in network or physical proximity to the client. The said device comprises: -means for determining the availability of said first set of content servers or load directors based on capacity and load conditions and providing the connection address of such content server or load director to the client if available, -means for providing the connection address of a content server or load director that is not in said first set if none of the first set content servers or load directors are available, -means for enabling customized content to said client in the case when connection to a content server or load director from said first set is possible, based on the information arising out of said network or physical proximity to said client. This invention also includes a method of providing improved data transfer speeds.
