Title of Invention

A METHOD FOR ACCESSING, VIEWING AND MANIPULATING DATA STORED IN A COMPUTER SYSTEM
Abstract A method for accessing, viewing and manipulating data stored in a computer system having one or more computers with at least one storage device, the method comprising: storing a set of entries in a permanent archive on one or more of the computers of the computer system; storing a collection of references to one or more of the entries in a notebook on one or more of the computers; displaying the collection of references and one or more entries in a window of one of the computers; manipulating the collection of references in the notebook; adding entries into a journal on one of the computers by creating a plurality of modifiable objects; storing the plurality of modifiable data that objects on the storage device of the computer system; storing the entries in the permanent archive by converting the modifiable data objects into non-modifiable data objects; searching the entries according to one or more new search parameters; in response to the search, creating one or more new references to the entries satisfying the search parameters and adding the new references to the collection of references; and wherein the first collection of references is stored in a first notebook on one of the computers and a second collection of references to the entries is stored on a second notebook on another computer, wherein the computers are connected and communicate over a network.
