Title of Invention

METHOD OF TAKING PRINTOUT OF PICTURES USING THE DETACHABLE LIQUID CRYSTAL DISPLAY
Abstract The invention explains a detachable LCD with memory and also explains a method of taking printouts of pictures using the detachable LCD display. According to the method, the detachable LCD is provided with an inbuilt memory space. The pictures taken from the camera are stored in this memory space of the LCD. The detachable LCD can then be connected to a printer using USB protocol, wherein the LCD acts a USB device and the printer acts as a USB host. The printer is provided with USB host implementation for this purpose. The LCD allows a user to navigate through the pictures stored in the memory space of the LCD and firing the printout of the desired picture.
Full Text FIELD OF THE INVENTION
The present invention, relates to the printers which are used for printing high graphics pictures. More particularly the present invention relates to detachable LCD with memory and a method of taking printout of pictures using the detachable LCD.
DESCRIPTION OF RELATED ART
There are Cameras which have tiltable, rotatable and detachable LCDs, These detachable LCDs communicate with Cameras through some protocol. Till date these LCDs are just used for easy handling and safety so that scratches can be prevented.
LIMITATIONS
These LCDs are not supported for being connected to any other device such as printers. Once user takes a picture from the camera there are mainly two ways through which user can take a print of his/her picture.
• Connect the camera to PC, transfer the file to PC and then print it
from printer connected to the PC
• Connect the camera directly to Printer through PicBridge and then
get the print. Figure 1 depicts the process of printing through a PC
and printing through a PicBridge

SUMMARY OF THE INVENTION
The main object of the invention is to provide a method of taking printouts of pictures using the detachable LCD display. Another object of the invention is to provide a detachable LCD with memory.
Accordingly, this invention explains a method of taking printouts of pictures using the detachable LCD display of a camera comprising the steps of:
(a) providing inbuilt memory space in the detachable LCD;
(b) storing the pictures taken in the memory space of the LCD;
(c) connecting the detachable LCD to a printer using USB protocol, wherein
the LCD acts as a USB device and the printer acts as a USB host;
(d) navigating through the pictures stored in the memory space of the LCD and
firing the printout of the desired picture to a printer.
The USB Host implementation is added to the printer.
Accordingly, this invention further explains a detachable LCD display of a camera comprising:
(a) an inbuilt memory space for storing the pictures taken using the camera; and

(b) navigation means to view the pictures stored in the memory space of the LCD to fire the printout of the desired picture to a printer.
These and other objects, features and advantages of the present invention will become more apparent from the ensuing detailed description of the invention taken in conjunction with the accompanying drawings.

The invention relates to the field of printers and cameras having detachable LCDs, and more specifically to a method of taking printouts of pictures using the detachable LCD display of a camera. According to the method, the detachable LCD is provided with an inbuilt memory space. The pictures taken from the camera are stored in this memory space of the LCD. The detachable LCD can then be connected to a printer using USB protocol, wherein the LCD acts a USB device and the printer acts as a USB host. The printer is provided with USB host implementation for this purpose. The LCD allows a user to navigate through the pictures stored in the memory space of the LCD and firing the printout of the desired picture.
In general there are Cameras, which have detachable LCDs, now the same Protocol with which the Camera is talking with LCD can be used in Printer case. So we can have the printers with detachable LCD. Figure 2 depicts detachable LCD, which can be connected to both Camera and Printer
Figure 3 depicts full process to achieve get the print. The steps involves:

• Taking a snap from camera which has detachable LCD and make
sure that you have selected to save it in the detachable LCD
memory.
• Remove the LCD from the camera.
• Attach the LCD to the printer.
• Select the picture using the printer keypad.
• After selecting the picture give the print.
Now in this case the LCD will have its own memory and all the pictures taken by camera will be stored in LCD memory if user has selected so as there are other types of memories also in Camera (SD Card, Camera's internal memory). Since LCD is also having its memory so that memory also has to communicate with the printer and for this communicate USB protocol can be used in which LCD will behave as USB device and Camera/Printer will behave as USB Host. For this we may need to add the USB Host implementation to the printers. Hardware can be designed in such a way that will have connector for LCD as well as connector for Memory (USB Connector).
Since the LCD size and communication protocol will be same for Camera and printers so user can easy scroll the pictures stored in the LCD while it's attached to the printer as it used to be while it's attached to the camera. At any point of time user can select any picture and can give the print command.

So through this we can easily click any picture, detach the LCD from the camera, connect to the Printer and can easily fire the print. This will help us in eliminating the PC (if PicBridge is supported at Printer end) in between which is at present always needed to get a print of any picture taken from the Camera.
The method of taking printouts of pictures using the detachable LCD display of a camera involves few steps which is illustrated below.
An inbuilt memory space in the detachable LCD is provided. The invention involves storing the pictures taken in the memory space of the LCD. For taking the printout the invention involves connecting the detachable LCD to a printer using USB protocol. The LCD acts as a USB device and the printer acts as a USB host. The LCD further enables navigating through the pictures stored in the memory space of the LCD and firing the printout of the desired picture to a printer. The USB Host implementation is added to the printer. The detachable LCD display of a camera involves an inbuilt memory space for storing the pictures taken using the camera and navigation means to view the pictures stored in the memory space of the LCD to fire the printout of the desired picture to a printer.
ADVANTAGES
1. A person need not carry Camera with him if he wants to take print
from the printer.
2. In USB print you cannot view the files on printer screen, where as in
this you can scroll, see and select and picture you want to print.

3. Normally PicBridge is used to transfer the pictures from camera to printer in order to print, but in this case no need for PicBridge connector.
