Title of Invention

"MORSE CODE TRAINER CUM TRANS RECEIVER"
Abstract The present invention provides a Morse code trainer cum Trans receiver system comprising: Morse key for sending Morse code, gating circuit connected between Morse key and programmed micro controller for stimulating micro controller, the output of micro controller is connected to audio pre-amplifier to enhance the audio signal level, the output of pre-amplifier is further connected to audio power amplifier, said pre-programmed micro controller is connected to LCD driver cum display unit for displaying the Morse code and its corresponding alpha numeric character for training purpose, said pre-amplifier output is also connected to an audio detector for conversion of audio signal into DC signal, a relay driver is connected to the output of the detector to activate the radio transmitter to transmit the Morse code, a FM radio receiver which receives the Morse code signal from the distant transmitter, said Morse code signal is attenuated to required level before connecting to Band Pass filter to pass the Morse code signal, and a phase lock loop circuit receives the Morse code signal from BPF and compares its self oscillating frequency before giving its output to gate circuit to stimulate the micro controller to display the Morse code signal as received from receiver through LCD display cum driver.
Full Text This invention relates to Morse code trainer cum trans receiver system.
BACKGROUND
In older days, Morse code communication was the most efficient means of communication. In Morse code communication, a Morse code trained person used to send messages by make and break key. The formation of Morse code constitutes dots and dashes of proper duration and spacing between them. But these Morse code operation differs from man to man due to the following existing drawbacks and thereby convey a wrong message.
Drawbacks:
the length of dots and dashes cannot be monitored
there is no facility to monitor the code and corresponding alpha/ numeric
display for the trainee,
spacing between character and group code cannot be monitored,
normally broken Morse code transmission / training was carried out
there was no facility for the conversion of Morse code into alpha / numeric
character
The object of this invention is to overcome the above drawbacks in training as
well as in Morse code communication by providing:
adjustable speed of sending messages by beginner to train operator
time duration for dot and dashes to be adjusted,
alpha / numeric display for the trainee for his corresponding Morse
monitoring facility for the supervisor / instructor for checking
error indication for wrong space and duration of Morse code
alpha-numeric display at receiver which facilitates the receiver to read the
Morse code messages in plane version

The other object of this invention is to provide a Morse code communication system at low cost and light in weight for deploying at remote places.
To achieve said objective this invention provides a Morse code trainer cum Trans
receiver system which comprises: -
Morse key for sending Morse code,
gating circuit connected between Morse key and programmed micro
controller for stimulating micro controller,
the out put of micro controller is connected to audio pre amplifier to
enhance the audio signal level,
the output of pre amplifier is further connected to audio power
amplifier,
said programmed micro controller is connected to LCD driver cum
display unit for displaying the Morse code and its corresponding
alpha numeric character for training purpose,
said pre amplifier output is also connected to an audio detector for
conversion of audio signal into DC signal,
a relay driver is connected to the out put of the detector to activate
the radio transmitter to transmit the Morse code,
a FM radio receiver which receives the Morse code signal from the
distant transmitter,
said Morse code signal is attenuated to required level before
connecting to Band Pass Filter to pass the Morse code signal, and
a phase lock loop circuit receives the Morse code signal from BPF
and compare its self oscillating frequency before giving its out put
to gate circuit to stimulate the micro controller to display the Morse
code signal as received from receiver through LCD display cum
driver
A power supply having input of 12 volt DC and the out put of +5 V and +12 Voltages respectively.
The speaker is connected to said pre amplifier for hearing the Morse code by the trainee.
The invention will now be described with reference to the accompanying drawings.
Figure 1 shows the Morse code trainer cum trans receiver system, according to
this invention.
Referring to the drawing, (1) shows AM / FM receiver, which receives the Morse coded messages from distant transmitter and converts it into audio signal in the form of Morse code. The AM / FM receiver (1) is connected to attenuator (2) to feed received Morse code signal to band pass filter (3) after a pre-determined level of attenuation of about 10 dB. The said band pass filter (3) has a sharp response for the modulated frequency and is connected to the phase lock loop circuit (PLL} (4), which decreases its output till the input frequency is equal to its oscillating frequency. Gate circuit (5) is connected to said PLL circuit (4) to provide interface between Morse key (13), phase lock loop circuit (PLL) (4) and the microcontroller (6). The microcontroller (6) is connected between gate circuit (5) and pre-amplifier (8) and is used as three port flash back. Port (P1) is used as output and interrupt zero (INTRP 0) is used as input for key control Circuit. The microcontroller (6) is programmed to have three outputs i.e. alphanumeric display, Morse code and audio output in accordance with the key control duration for dot and dash can be adjusted through control port of the microcontroller. The microcontroller is^Pre Programmed as per the requirement such as:-
a) Display each alphanumeric character.
b) Display corresponding Morse code.
c) Display time duration in millisecond.
d) Display speed of the Morse.
LCD Drive and display circuit (7) consists of in-built micro-chip and LCD display, which are having two row 16 display character. A fixed 1 KHz audio frequency tone is fed to microcontroller (6) by keying as per the time duration for dot and dash. The microcontroller (6) will generate code and also display the character. The Morse code from the microcontroller (6) is fed to pre-amplifier (8). The output of said pre-arnplifier is connected to AFA (9) and to detector (10). It amplifies the audio signal before the reproduction of audio into Morse Codes. The detector / relay driver (10)/10a receives audio signal and converts into a bias signal for the relay driver 10 (a). The relay driver (10)a is designed for adjustable time delay, which helps the transmitter (11) to be ON for the same period of time duration for which the Morse Code is being sent. The transmitter (11) can be any one ie. AM / FM. The audio Morse Code is further modulated over carrier frequency and transmitted. The power supply unit (12) accepts 12 Volts DC as input and provide 5 Volts and 12 Volts, as output for the system to operate. The Morse Key (13) is for sending Morse code during transmission only and trainee can use it for training.

driver cum display unit (7) for displaying the Morse code and its
corresponding alpha numeric character for training purpose;
said pre-amplifier output (8) is also connected to an audio detector
