Title of Invention

ISSUING A DIGITAL RIGHTS MANAGEMENT (DRM) LICENSE FOR CONTENT BASED ON CROSSFOREST DIRECTORY INFORMATION
Abstract An organization maintains a computer network comprising at least a forest A and a forest B, where forest A has a directory A and a querying entity A capable of querying directory A, and forest B has a directory B and a querying entity B capable of querying directory B. Querying entity A receives a request from the user and decides whether to grant the request based at least in part on whether the user is a member of the group. Thus, querying entity A queries directory A to return information on the group, is directed to forest B, contacts querying entity B, requests querying entity B to query directory B whether the user is a member of the group, receives a response, and grants the request from the user based at least in part on whether the user is a member of the group.

TECHNICAL FIELD
This invention relates to a digital rights management (DRM) system.
More particularly, the invention relates to employing a DRM system to publish
digital content in an organization such as an office or corporation or the like such
that rendering and use of the content within the organization may be constrained
according to corresponding use or license terms. Even more particularly, the
present invention relates to issuing a digital license for corresponding digital
content based on group information obtained from a directory from another forest.

SUMMARY OF THE INVENTION
The aforementioned needs are satisfied at least in part by the
present invention in which it is determined whether a user from an organization is
a member of a group pre-defined within the organization, where the organization
maintains a computer network comprising at least a forest A and a forest B, where
forest A has a directory A and a querying entity A capable of querying directory A,
and where forest B has a directory B and a querying entity B capable of querying
directory B.
In the method, querying entity A receives a request from the user
and decides whether to grant the request based at least in part on whether the
user is a member of the group. Thus, querying entity A queries directory A of
forest A to return information on the group and in response is directed to forest B.
Querying entity A therefore contacts querying entity B of forest B and requests
querying entity B to query directory B whether the user is a member of the group.
Querying entity A receives in response from querying entity B whether the user is
in fact a member of the group, and grants the request from the user based at least
in part on whether the user is a member of the group.
