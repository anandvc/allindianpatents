Title of Invention

A METHOD FOR DYNAMIC LOCATION MAP PRINTING IN A MULTI FUNCTIONAL PERIPHERAL (MFP) DEVICE
Abstract The invention in general relates to the field of Dynamic Location Map Printing through MFP. This invention explains a method for dynamic location map printing in MFP comprising the steps of: choosing a source MFP by the user; selecting the location MAP printing option in said source MFP; giving a valid telephone number to which a destination MFP is connected as input; initiating a FAX call to the said telephone Number to which said destination MFP is connected; getting the location details through the Non Standard Facility and disconnecting the call; sending the location details to a Location Server by the source MFP; sending to the said source MFP by the server the part of MAP considering the source MFP as start point and the specified Location Details as END point;and receiving the DATA MAP by the MFP and Prints the information.

FIELD OF INVENTION
This invention, in general, relates to the field of multifunction peripherals (MFP). Further this invention relates to additional services / features such as location based map printing in MFPs. More particularly, this invention relates to a method for dynamic location mar printing in MFP.
DESCRIPTION OF RELATED ART
United States Patent number 6,081,609 titled “Apparatus, method and medium for providing map image information along with self-reproduction control information” is incorporated herein to explain the present state of technology.
The invention proposed in the said patent enables to provide route information obtained by a search, even in a personal computer not provided with the same application software. The said invention includes: a route calculation module for creating a predetermined sequence of image information representing a route connecting at least two points of a starting point and a destination; a navigation guidance creation module; and a self-reproduction file creation module for creating a reproduction control data for reproducing the predetermined sequence of image information and filing the image information and the reproduction control data.
The currently available technology does not provide for dynamically tracking and finding out a destination point from the current position and Map Printing based on the same.
SUMMARY OF THE INVENTION
The corporate building becomes very huge and it's hard to track a particular location or to find an easiest way to reach the particular place inside and as well as outside the building. In such cases it becomes easier to find out the location or meeting points from where the current position by printing the location map to the destination from a MFP. The invention is to dynamically spot and print a location
2




map through MFP. MFP is hosted at various places inside a corporate building. It can be done by the following steps.
Accordingly, this invention explains a method for dynamic location map printing in MFP comprising the steps of:
(a) choosing a source MFP by the user;
(b) selecting the location MAP printing option in said source MFP;
(c) giving a valid telephone number to which a destination MFP is connected as input;
(d) initiating a FAX call to the said telephone Number to which said destination MFP is connected;
(e) getting the location details through the Non Standard Facility and disconnecting the call;
(f) sending the location details to a Location Server by the source MFP;
(g) sending to the said source MFP by the server the part of MAP considering the source MFP as start point and the specified Location Details as END point; and
(h) receiving the DATA MAP by the MFP and Prints the information.
The telephone number is the nearest location towards the destination. The Location Server hosts all Maps. The source MFP provides a select location map printing option through generic user interface.

DETAILED DESCRIPTION OF THE INVENTION
The preferred embodiments of the present invention will now be explained with reference to the accompanying drawings. It should be understood however that the disclosed embodiments are merely exemplary of the invention, which may be embodied in various forms. The following description and drawings are not to be construed as limiting the invention and numerous specific details are described to provide a thorough understanding of the present invention, as the basis for the claims and as a basis for teaching one skilled in the art how to make and/or use the invention. However in certain instances, well-known or conventional details are not described in order not to unnecessarily obscure the present invention in detail.
The MFP is generally hosted at various places in corporate buildings. The invention is to find the location details of a destination MFP and to print the route MAP to the destination MFP. A network MAP server will host the location maps to various locations also the route maps from the source to destination.
The user chooses one MFP. The user then has to select the location MAP printing option in MFP, and gives a valid Telephone number (to which destination MFP is connected) as input. The Telephone number is the nearest location towards the destination. Then the MFP Initiates a FAX call to the Specified Telephone Number (to which destination MFP is connected) and gets the Location Details through the NSF (Non Standard Facility), then disconnects the call. The MFP then sends the Location Details to the Location Server. The Location Server hosts all Maps. The Location Server sends the MFP the part of MAP considering the MFP as start point and the specified Location Details as END point. The MFP receives the DATA MAP and Prints the same.
The MFP(s) source MFP provides a select location map printing option through generic user interface, step 1 of figure 1. The MFP(s) prompts the user to input the destination MFP, MFP (d) telephone number, step 2 of figure 1. The MPF(s) initiates a FAX call to the destination MFP, step 3 of figure 1. MFP(s) gets location details of MFP (d) by the NSF filed in the FAX initiation call procedure. Step 4 of
