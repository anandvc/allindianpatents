Title of Invention

''AN AUTOMATIC DELAYED RELEASE CLEANER DISPENSER SYSTEM FOR WCs.''
Abstract An automatic delayed release cleaner dispenser system for WCs characterised in that is a container (1) adapted to be secured with the frame (3) to be provided in the flushing cisterns, a dispenser cup (5) being provided at the bottom of said container (1), so as to supply the cleaning/deodorant to the flushing cisterns, a float (7) being provided with said stand so as to actuate the cup (5) release mechanism provided below said dispenser cup (5), adjusting screws being provided at the bottom of said stand so as to adjust the height of the stand in the flushing cisterns, an inlet (12) being provided near the top end of said container (1) for supplying cleaning/deodorant mixture into the container (1).
Full Text I CLAIM;
1. An automatic delayed release cleaner dispenser system for WCs
characterised in that is a container (1) adapted to be secured with the
frame (3) to be provided in the flushing cisterns, a dispenser cup (5)
being provided at the bottom of said container (1), so as to supply the
cleaning/deodorant to the flushing cisterns, a float (7) being provided
with said stand so as to actuate the cup (5) release mechanism
provided below said dispenser cup (5), adjusting screws being
provided at the bottom of said stand so as to adjust the height of the
stand in the flushing cisterns, an inlet (12) being provided near the
top end of said container (1) for supplying cleaning/deodorant
mixture into the container (1).
2. An automatic delayed release cleaner dispenser system as claimed in
claim 1 wherein said container is secured with the legs (2) at the
frame (3) by making use of securing clamps (4).
3. An automatic delayed release cleaner dispenser system as claimed in
claim 1 wherein said cup release mechanism comprises a main lever
(6) adapted to be secured pivotaly with the frame (3), a release lever
(6a) being provided with the other end of said main lever so as to be
actuated by the help of float (7) provided with said frame (3).
4. An automatic delayed release cleaner dispenser system as claimed in
claim 3 wherein a rubber seal (9) is provided at the top surface of said
main lever to stop the leakage cleaning/deodorant leakage from the
dispenser cup.

5. An automatic delayed release cleaner dispenser system as claimed in
claim 4 wherein a spring (8) is provided between said main lever (6)
and the leg (2) of the frame (3) so as to press the main lever (6)
towards the mouth of the dispenser cup (5).
6. An automatic delayed release cleaner dispenser system as claimed in
claims 1 to 5 wherein a trigger lever (13) is secured with the float lever
of the cisterns so as to actuate the release lever of the cup release
system to supply the cleaning/deodorant to the flushing cisterns
when the flushing operation is near to be completed.
7. An automatic delayed release cleaner dispenser system substantially
as herein described and illustrated.
