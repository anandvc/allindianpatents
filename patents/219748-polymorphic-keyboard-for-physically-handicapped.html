Title of Invention

POLYMORPHIC KEYBOARD FOR PHYSICALLY HANDICAPPED
Abstract POLYMORPHIC KEYBOARD FOR PHYSICALLY HANDICAPPED ABSTRACT In order to have high-tech life with the computer for a handicapped person in his/her day to day activities, communication with others, research work, entertainment and in many fields, a polymorphic keyboard has been thought of to interact with the computers as one has to move his hand over the normal keyboard with mouse which is painful for the handicapped. With the mouse, it is still difficult for him to perform. This designed polymorphic keyboard, that is four times shorter than the normal keyboard in the computers has been designed to enable a handicapped person to operate without strain. The keyboard that has been designed is polymorphic which provides a solution to have access to computers using an economical keyboard for the single hand, single finger handicapped persons and persons in the wheel chair, allowing them to explore the opportunities offered by information technology. It is a single handheld device and can be operated with a single finger. This device has 16 keys which brings into the entire operation of 104 computer keyboard. It can be used to control 5 appliances simultaneously and to interact with the personal computer.

DESCRIPTION
(1) OBJECTIVE OF THE INVENTION;
The objective of the invention is to give comfort to the physically handicapped persons, such as those with the loss of one hand or persons, who can move only in the wheel chairs. By providing them with a "Multifunctional wireless remote keyboard" which is capable of giving input/output to the distant computer, which can control at the maximum of five electrical devices such as light (No.6 as shown in figure 1), tape recorder (No.5 as shown in figure 1), Television(No.7 as shown in figure l)or any commercial equipment like billing machine around the person, ergonomically designed for the physically handicapped.
A polymorphic keyboard designed with 16 keys (No.l as shown in figure 1) compactly is used here and this acts as a remote controller. In this keyboard, each key performs multiple functions based on the mode of operation. Shortcut functions can also be operated with predefined keys in the remote keyboard. This keyboard operates using Frequency Modulation for communication with the distant computer.
This polymorphic keyboard has the enhanced virtual mode where desktop shortcuts, small sequence of operations required to be done can be configured and triggered by pressing a single button. This is a single hand operated wireless device and it can be operated by a single finger.
No additional add on card is necessary. All controls and input operations can be performed with the personal computer and is compatible with most of the softwares like Word Pad, Power Point, and Excel that run on the windows platform. Peripheral Interface Controller and Visual Basic are used for the implementation of this operation. It is compact and hence cost is low.
This novel and fully functional remote keyboard for physically handicapped which can control the electrical devices around the person, has been invented for the first time.

(5) USING POLYMORPHIC KEYBOARD;
Selecting the TEXT Mode and CONTROL mode respectively (shown in fig. 3) does the typing and control operation of the computer. (6) CONSTRUCTION OF THE DEVICE;
This Polymorphic Keyboard device consists of three modules:
I) TRANSMITTER MODULE;
This module (as shown in figure 2) consists of Compact Keypad consisting of only 16 keys. The key pressed is encoded using the DTMF encoder and is fi'equency modulated for transmission using FM Transmitter.
II) RECEIVER MODULE;
The FM Receiver in this module (No.3 as shown in figure 1) receives the transmitted signal. The received signal is decoded by the DTMF decoder . The decoder output is provided as Input to the PIC Microcontroller This microcontroller converts the input 4 bit hexadecimal code into an 8 bit ASCII code and send signals to the serial port of the computer or to the Appliances On/Off unit to perform the task desired by the user.
III) DEVICE DRIVER MODULE;
This module (No.2 as shown in figure l)is coded using Visual Basic 6.0 to implement the operating modes according to the requirement of the user. This reads the serial port and issues appropriate messages to the application in the Windows platform to get the desired result of typing/control. This module can be extended to provide a Virtual mode in which the user can assign shortcuts to commands for each application, assign long sentences to a single key, etc

(7) EXECUTION OF THE COMMAND BY DEVICE DRIVER MODULE
The key pressed is interpreted and the symbol is identified. The mode of operation of the keyboard is based on the mode selection word. The scan code for the key pressed and the desired symbol is assigned according to the mode which is currently selected. The scan code is placed in the keyboard buffer to execute the key pressed.
