Title of Invention

"A DEVICE FOR PERFORMING AUTHORIZATION OF AN IGNITION KEY"
Abstract A system for performing authorization of an ignition key comprising an engine control unit (ECU), an ignition key that stores a key ID, a lock password and a key password; and a transponder provided within the ignition key , wherein the ECU performs the steps S210 to S240 of: 1) receiving the key ID from the ignition key and determining if the Key ID is a registered ID; and 2) generating, if the key ID is the registered ID, a random number and encoding a stored lock password using the random number, and transmitting the random number and the encoded lock password to the transponder of the ignition key , and accordingly the transponder performs the steps S242 to S270 of: 3) decoding the lock password using the received random number and encoded lock password, then determining if the decoded lock password is identical to a stored lock password; and 4) encoding a key password using a stored key password, and transmitting the encoded key password to the ECU, and accordingly the ECU performs the steps S272 to S290 of: 5) decoding the received encoded key password, then determining if the decoded key password is identical to the stored key password; and 6) releasing an ignition lock state if the decoded key password is identical to the stored key password.
