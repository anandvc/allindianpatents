Title of Invention

DOWNLINK BEAM FORMING METHOD AND DEVICE OF TIME DIVISION CODE DIVISION MULTIPLE ACCESS SYSTEM
Abstract A downlink beam forming method of time division code division multiple access system, the method includes: the space covariance matrix of all the uplink timeslot interfere user signals and the downlink timeslot that exist the interfere user signal are obtained (S301); the interfere space covariance matrix of each downlink timeslot is obtained according to the downlink timeslot that exist the interfere user signal and the space covariance matrix of the interfere user signal (S302); the beam forming weight coefficient of downlink desired user signal is determined according to said interfere space covariance matrix of each downlink timeslot and the space covariance matrix of the downlink desired user signal (S303); the beam forming is performed according to the beam forming weight coefficient of downlink desired user signal (S304). The present invention can calculate the interfere space covariance matrix of all the downlink timeslot by determining the downlink timeslot that exist all the interfere user signal, and then obtain the beam forming weight system of all the downlink desired user signal, so as to perform beam forming, so the beam forming method of the present invention is the same with the condition of timeslot symmetry or un-symmetry.
