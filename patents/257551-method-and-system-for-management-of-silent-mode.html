The following Specification describes the nature of the invention
FIELD OF INVENTION
The present invention, in general, relates to the field when a mobile user is in silent mode and could not notice an incoming call hence misses it. More particularly the present invention relates to a method of informing the caller about the called party silent mode and the silent mode is scheduled based to indicate the user about the mode and other information that mobile has got during that particular mode.
DESCRIPTION OF RELATED ART
In the existing scenario a mobile user may put the phone in silent mode in the office or in some meeting. Sometimes the user forgets to put it back in normal mode (for e.g. even after going home) and misses some calls. It is quite possible that the user can miss some very important calls.
The caller doesn't know why the called party is not responding. The caller just knows that there is “No Response” from the called party.
There is no technology to
§ Indicate to the caller that the called party is in silent mode
§ Enable the called party to be alerted in silent mode under some conditions,
based on scheduler and other possible ways.

FIELD OF INVENTION
The present invention, in general, relates to the field where a mobile user is in silent mode and cannot notice an incoming call, hence misses it. More particularly the present invention relates to a method of informing the caller about the called party silent mode and the silent mode is scheduled based to indicate the user about the mode and other information that mobile has got during that particular mode. More particularly, the present invention relates to a method and system for automatic management of silent mode.
DESCRIPTION OF RELATED ART
In the existing scenario a mobile user may put the phone in silent mode in the office or in some meeting. Sometimes the user forgets to put it back in normal mode (for e.g. even after going home) and misses some calls. It is quite possible that the user can miss some very important calls.
The caller doesn’t know why the called party is not responding. The caller just knows that there is "No Response" from the called party.
There is no technology to
§ Indicate to the caller that the called party is in silent mode
§ Enable the called party to be alerted in silent mode under some conditions, based on scheduler and other possible ways.
SUMMARY OF THE INVENTION
The invention proposed herein is applicable in a scenario where a mobile user is in silent mode and cannot notice an incoming call, hence the user misses the call. Further, the proposed invention would provide for informing the caller about the called party silent mode and the silent mode is scheduled based to indicate the user about the mode and other information that mobile has got during that particular mode.
Accordingly, this invention explains a method for automatic management of silent mode comprising the steps of: On call terminator side:
(a) setting the mobile in the silent mode by the terminator and configuring the maximum “n” rings for sending an alert message;
(b) getting the incoming call from originator by the terminator;
(c) proceeding the call in normal way if the terminator attends the said call;
(d) filling the alert in the User-User Information element of the CC Message if the terminator does not attend the call after the set “n” rings;
(e) sending the alert message by the terminator to the originator;
On call originator Side:
(f) receiving the User-User information in CC Message by the originator; and
(g) displaying the said alert message to the originator.

The terminator is given an option to put the phone in silent mode for a certain period of time and after the expiry of that time the phone will either return to normal mode or prompt the terminator to take a necessary action. The alert message can be generated after specific repeated missed calls. The alert message informs the originator that the terminator is in silent mode. The duration and continuation of the said silent mode can be configured when the terminator selects the option for entering the silent mode.
Accordingly, this invention further explains a system for automatic management of silent mode comprising: On call terminator side:
(a) means for setting the mobile in the silent mode by the terminator and configuring the maximum “n” rings for sending an alert message;
(b) means for getting the incoming call from originator by the terminator;
(c) means for proceeding the call in normal way if the terminator attends the said call;
(d) means for filling the alert in the User-User Information element of the CC Message if the terminator does not attend the call after the set “n” rings;
(e) means for sending the alert message by the terminator to the originator;
On call originator Side:
(f) means for receiving the User-User information in CC Message by the originator; and
(g) means for displaying the said alert message to the originator.

These and other objects, features and advantages of the present invention will become more apparent from the ensuing detailed description of the invention taken in conjunction with the accompanying drawings.
