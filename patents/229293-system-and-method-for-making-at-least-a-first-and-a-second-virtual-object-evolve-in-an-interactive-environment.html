Title of Invention

SYSTEM AND METHOD FOR MAKING AT LEAST A FIRST AND A SECOND VIRTUAL OBJECT EVOLVE IN AN INTERACTIVE ENVIRONMENT
Abstract The present invention concerns a system and a method for making a virtual object (150) evolve in an interactive environment. According to the present invention, this system includes at least a first portable unit (2; 2.1, 2.1', 2.2, 2.3) and at least a first computer terminal (1; 1.1,1.2, 1.3) each including wireless transmitting and receiving means (26) for transmitting data relating to said virtual object (150) from the portable unit to the computer terminal, or respectively from the computer terminal to the portable unit, so that said virtual object (150) is capable of evolving, either in a first so-called main interactive environment (100), supported by the computer terminal (1; 1.1,1.2, 1.3), or, in a second so-called secondary interactive environment (200), supported by the portable unit (2; 2.1, 2.1 ',2.2,2.3).
