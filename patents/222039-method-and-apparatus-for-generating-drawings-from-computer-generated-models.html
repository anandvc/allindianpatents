Title of Invention

METHOD AND APPARATUS FOR GENERATING DRAWINGS FROM COMPUTER GENERATED MODELS
Abstract A METHOD AND APPARATUS FOR GENERATING AT LEAST ONE DIGITAL DRAWING USING A SERVER SYSTEM INCLUDING A CLIENT SYSTEM HAVING A BROWSER AND A DATA STORAGE DEVICE, THE SERVER SYSTEM COUPLED TO THE CLIENT SYSTEM AND THE DATA STORAGE DEVICE, A PLURALITY OF STANDARDIZED ORTHOGRAPHIC RULES AND A COMPUTER GENERATED MODEL OF PART STORED IN THE DATA STORAGE DEVICE AND ACCESSIBLE BY THE SERVER SYSTEM, SAID METHOD COMPRISING THE STEPS OF : GENERATING A PLURALITY OF DIGITAL DRAWINGS VIEWS WITH THE SERVER SYSTEM, SUCH THAT THE DIGITAL DRAWING VIEWS ARE BASED ON THE PRE-STORED STANDARDIZED PROJECTION RULES AND THE COMPUTER GENERATED MODEL; AND CHARACTERIZED BY USER EDITING THE PLURALITY OF DIGITAL DRAWING VIEWS WITH THE SERVER SYSTEM THROUGH THE CLIENT SYSTEM.
