Title of Invention

A METHOD FOR PROVIDING ONE TOUCH DIAL FACILITY FOR FAX, SCAN TO EMAIL OR INTERNET FAX IN MULTI FUNCTION PERIPHERALS
Abstract The present invention provides a 'one touch dial method' to the users of MFP for fax or Scan to Email or Internet Fax in MFPI AIO without the need of a dedicated Keypad. The user may configure any number for dialing through any of the keys in the keypad. One dedicated key will be used for enabling one touch operation and this key switches the machine between one touch mode and normal mode. In addition an LED is provided for indicating that the machine is in one touch mode. Moreover this method helps in minimizing the size of MFP/AIO owing to the efficient usage of the keys without requiring a dedicated key pad for one touch operation.

FIELD OF THE INVENTION
The present invention relates in general to the Multi function Peripherals (MFP). Further, this invention relates to "one touch dial" facility for fax, Scan to Email or Internet Fax in Multi Function Peripherals (MFP) / All in One (AIO). More particularly, the invention relates to a method for providing one touch dial facility for fax, scan to Email or Internet Fax in multi function peripheral devices.
DESCRIPTION OF RELATED ART
In order to increase user convenience, MFP / AIO devices are provided with various easy and convenient dialing methods. One Touch Dial is one among those easy and quick dialing methods. One touch dial means dialing of a fax/telephone on one touch of a key. Ordinarily, a dedicated keypad will be attached with the MFP / AIO to achieve this dialing facility and a number will be stored in the memory corresponding to each key of the keypad. This implies that if the device has to support 50 one-touch numbers, then it would require 50 dedicated keys with the machine and hence comes with the cost of 50 additional keys.
However alternatives are available in the art for conferring speed dialing without using a dedicated keypad. It is found in the user guide of OKIFAX 5750 MFP (http://vtel.pageprocessor.nl/import/bestanden/pdf/Gebruiksaanwijzing_Oki_fax_5 750_eng.pdf) that it can facilitate speed dialing (one-touch or 2 and 3 digit speed dial numbers) for multiple faxing phone numbers. This MFP device provides a dedicated speed-dial key for actuating the speed-dial mode so that a user can press the dedicated speed-dial key followed by the associated digit to connect to a desired address and the device includes an LCD panel that indicates all functions and features during operation.
A facsimile device capable of sending e-mails messages as well with speed dialing
is available in the related art. For example Panasonic KX-PW88CL Facsimile
device along with a Cordless Phone
(http://panasonic.co.jp/pcc/cs/en/telecom/otakkusu/PW88CL_English_Quick_Guid

e.pdf) comprises a dedicated 'Speed Dial' button for activating the speed-dial mode so as to enable the user to connect to a desired address by pressing the dedicated speed-dial key followed by the associated digit. Further, this facsimile device includes an LCD panel that indicates all functions and features during operation. Another example with the same feature is Sharp UX-5000 Facsimile device (www.sharpusa.com/files/fax_man_ux5000.pdf), which has a dedicated speed dial button for actuating the speed-dial mode. A user can press the dedicated speed-dial key followed by the associated digit to connect to a desired address. However the operation manual of the above mentioned product does not mention whether e-mail addresses etc. can be programmed in the one touch-dial feature.
The state of the art is also reflected in the patent literature US Patent Number US 4,567,326 titled "Telephone Apparatus for Selective Operation in a Normal and Repertory Dial Mode" which claims a telephone apparatus comprising ten keys for regular dialing and two special keys in which one special key is used to switch the telephone in an abbreviated dial mode. Each of the ten regular dialing keys may be associated with different telephone numbers. Once the telephone is switched to abbreviate dial mode, it can be connected to other telephones through one-touch dialing. But application of this device is limited only to telephone apparatus and nothing is mentioned if e-mail addresses etc. can be programmed. Similarly European Patent Number EP0863659B1 (Filed March 03, 1998) titled "User Friendly Interface for a Facsimile Machine Integrating Programmed Dialing and Monitor Dialing" assigned to Hewlett-Packard Company relates to a facsimile machine that provides programmed dialing and speed dialing features. The facsimile machine comprises a dedicated 'Speed Dial' button on its front panel and is used to invoke the speed dial function and to select between single and double-digit entries. United States Patent numbers 5,427,318, 5,491,745, 5,509,067 and 5,838,784 are also included herein by reference.
US patents 5,427,318 describes a one-touch on-off showerhead valve apparatus for controlling the flow of fluid by pushbutton operation that includes a body having

an inner cavity which is divided into first, second and third spaces by a bellows and a support plate mounted therein.
The US patent 5491745 as per the description states that it tries to obviate the need for additional function keys or multiple step key activation. It also provides only twelve standard keypad keys. If further additional memory is needed it prescribes the use of additional function keys contrary to the earlier description.
US patent 5,509,067 describes a dial calling method in which a multiplicity of dialing information items can be registered while decreasing the number of keys and simplifying the operation and which is capable of abbreviated-dialing and one-touch-dialing a multiplicity of called parties in accordance with the registered dialing information, and to a communication apparatus which use the method.
US patent 5,838,784 describes a one-touch dial apparatus and method for registering dialing numbers in a telephone system includes a memory module divided into a plurality of memory blocks, each memory block having a number of memory locations, wherein each of the memory blocks is arranged for each of the corresponding telephones.
Even as the one touch concept is there in the above patents the present invention
on the other hand solves the problem in a better way. One dedicated key will be
used for invoking the one touch operation and this key will be used to toggle the
keypad between normal mode and one touch mode.
Also prior art devices do not feature the toggling of keyboard as presented in this
proposed invention where the whole keyboard changes its behavior for one touch
mode.
While the above referenced devices in the prior art accomplish their respective tasks, the application in a wide sense is limited for a number of reasons. Those devices either do not have dedicated key for one touch operation or they do not

have the facility to invoke an email address associated with the key in one touch mode.
It is also a drawback the device does not have an indicator like LED to indicate one touch mode. Also these devices do not feature the toggling of keyboard as presented in this proposed invention where the whole keyboard changes its behavior for one touch mode. Therefore based upon the foregoing there has been found a need to provide an MFP/AIO device having the facility one touch dialing using any of the keys available in the standard keypad.
SUMMARY OF THE INVENTION
The primary object of the present invention is to provide a 'one touch dial method' to the users of MFP for fax or Scan to Email or Internet Fax in MFP/ AIO without requiring a dedicated Keypad.
It is another object of the invention to provide a method wherein one dedicated key will be used for invoking the one touch operation and this key will be used to toggle the keypad between normal mode and one touch mode.
It is yet another object of the invention to provide indicating means such as LED to indicate to the user that the machine is in one touch mode.
It is a further object of the invention to provide a method, which enables the user to configure any destination number or Email address for dialing through any of the standard keys in key pad as one-touch key.
It is also an object of the invention to provide an MFP device that is cost effective, minimum in size and having the features for quick setting of operations such as storing Email address in Scan to Email application and Telephone Numbers or Addresses for Internet Application in MFP.

Accordingly, the invention comprises a a method for providing one touch dial facility in a multi function peripheral device having a keypad with a dedicated key for switching between a one touch dial mode and a normal mode and having an indicator for the indication of the said one touch mode, characterized in that the said mode is switched on when the dedicated key is pressed, and if the said dedicated key is held for a predetermined time enable a user to configure a one touch number where the number associated with a key is dialed when the key is pressed and released and the said device is shifted back to a normal mode when the said dedicated key is pressed while the device is in one touch dial mode.
These and other objects, features and advantages of the present invention will become more readily apparent from a reading of the following detailed description taken in conjunction with the drawings.

Now referring to figure 1, the operation of the one touch dial method according to the prior art is described. A key is dedicated for a particular number and the operation includes configuring it and dialing the key. As it is apparent from the figure, the following steps are to be followed for one touch dial method.
In the present invention of one touch dial method, a number can be stored corresponding to each key selected during configuration and upon single press of any key in key pad in one-touch mode, corresponding number will be dialed. The invention provides this 'one touch dial method' to the users of MFP for fax or Scan to Email or Internet Fax in MFP/ AIO without the need of a dedicated Keypad. A dedicated key will be employed for toggling the keypad between the normal mode and one touch dial mode. One LED is used for indicating if the full keypad is in one touch mode The dedicated key and the LED are new in both form and usage in this invention. Third part i.e. other keys are same as that used in the prior art but required for normal mode operation as well. The user may configure any number for dialing through any of the one-touch keys.
The method of performing the invention is follows.
When the dedicated key is pressed by the user, the MFP enters "one touch mode"
with the keypad ready for one touch mode operation and the status is indicated by

the LED. Any of the keys can be configured for a dial number by pressing and holding the key for a few seconds. If the key is not holding as said above, dialing process will be initiated. When dedicated key is pressed while operating in one touch mode, it goes back to normal mode and the LED is put to an off condition. Figure 2 shows the flowchart for explaining the operation of the invention featuring one touch key configuration. It shows that user needs to press one touch key once and then can configure as many numbers as required and on one touch of the key user can dial that number in one touch mode. Figure 3 and Figure 4 shows how to enter one touch mode and normal mode by using one touch mode key. When one touch key is pressed then led glows which means that machine is in one touch mode. In One touch mode, key pad will be toggled i.e. key will act as one touch key.
Figure 5 explains how the toggled key pad is handled. We need to add One Touch Handler module in existing systems to handle this toggled key pad concept. We can explain the sequence of operation as follows:
1. Key is pressed in key pad.
2. One Touch Handler is called.
3. One touch handler will invoke normal operation for the key if machine is in normal mode.
4. One touch handler invokes one touch operation if machine is in one touch mode i.e. one touch led is glowing.

A detailed description of the operation of the device according to the invention is given below.
Configuration: The following steps should be followed to configure any one-touch number.
1. Check whether device is in One touch mode i.e. One touch led is glowing or not.
2. If device is not in one touch mode then press one touch key to bring the device in one touch mode.
3. Hold any key for more than 2 seconds.
4. User would be asked to enter required information for one touch e.g. one touch number, one touch id etc,
5. If one touch key is configured already then to menu for replace or not existing entry would be displayed.
One Touch Dialing: steps given below should be followed to configure any one-touch number:
1. Check whether device is in One touch mode i.e. One touch led is glowing or not.

2. If device is not in one touch mode then press one touch key to bring device in one touch mode.
3. Press the respective key.
4. If one touch number is configured corresponding to that key then that number would be dialed.
5. If one touch number is not configured, then not configured status message would be displayed on LCD
The present invention provides many advantages compared to the existing art. One advantage is that the device size is minimized since the dedicated key panel is not required. Also this method can be used to store any quick setting. For example, Email addresses can be stored in Scan to Email Application and Fax numbers or email id can be stored for internet fax. It is also an achievement that the method reduces hardware cost, as only one key and one indicator need to be provided at key panel for one touch dial. Also as many as one touch numbers can be configured as many keys present in key pad.
Suppose there is a MFP device as known in the prior art with 100 keys on its key pad and 40 keys out of 100 keys were dedicated for one touch dialing. In that case, only 40 numbers can be configured for one touch dialing. That means only as many as one touch number can be configured as the number of dedicated one touch keys present in the key pad.
In the device of the present invention, one dedicated key for setting\resetting one touch mode has been added and all 40 dedicated one touch keys in previous art have been done away with. So, in comparison, now total of 61 keys (100-40 + 1) are present in the keypad of the MFP device instead of 100. All 60 keys except one

touch mode key can be used for one touch dialing facility whereby 60 numbers can be configured as one touch numbers and 40 keys can be saved in the key pad.
