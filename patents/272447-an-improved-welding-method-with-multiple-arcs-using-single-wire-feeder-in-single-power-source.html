AN IMPROVED WELDING METHOD WITH MULTIPLE ARCS USING SINGLE WIRE FEEDER IN SINGLE POWER SOURCE
Abstract The invention relate to improve the technology in submerged arc welding process, for feeding the wires from the two wire spools using two integrated wire feeding system with single motor shaft and power delivering from one power source with one control system is required fro performing the multiple arcs using single wire feeder in single power source in submerged technology. With new design of wire feeder simultaneously multi wires (same/ different) chemistry can be feed in the same wire feeder in same feed rate for different current rating. This new multiple arc using single wire feeder in single power source in submerged arc welding technology will enhance the production rate in double when compared to the conventional single wire submerged arc welding process. An integral shaft is designed and interfaced with the available DC wire feeder for coupling the additional wire feeder for feeding the wire in the separate arc cavity in SAW process. With this new multiple arcs using single wire feeder in single power source technology additional welding can be provided in both direction of welding. This results additional weld deposit with different heat in¬put energy available in the arc zone. Additional power source and control system is not required for this new design of multiple arcs using single wire feeder in single power sources in submerged arc welding technology. This new multiple arcs using single wire feeder system technology has interfaced with the existing SAW machine for plate welding, panel welding. In new multiple arc using single wire feeder in single power sources in submerged arc welding technology, a newly designed wire feeder (R), tine main wire, and a additional wire feed simultaneously using to different feeder assemblies and nozzles. The wires in the contact nozzles are arranged in parallel to the welding direction. In conventional SAW process the heat input energy was calculated by the separate arc energy of two different DC arcs. This new multiple arcs using single wire feeder in single power source in submerged arc welding technology is one of the advanced technique make use of single power source make two different arc energy available in the arc stream can increase the deposition rate as equivalent to the that of conventional SAW process. Thus, the new multiple arcs using single wire feeder in single power source in submerged arc welding technology can assure for increased transfer efficiency comparatively with other method. This results in a highest usable melting rate for given arc energy for good penetration, nil arc blow. Thus this multiple arcs using single wire feeder in single power source in submerged arc welding technique will results in 40 % increase in deposition rate and better mechanical properties are achievable over conventional Tandem submerged arc welding process.
