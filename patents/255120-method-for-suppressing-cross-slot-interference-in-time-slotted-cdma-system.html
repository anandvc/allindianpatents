Title of Invention

METHOD FOR SUPPRESSING CROSS-SLOT INTERFERENCE IN TIME-SLOTTED CDMA SYSTEM
Abstract A method for restraining cross-slot interference in slot CDMA system includes the following steps: BS determining the location of exsiting cross-slot according to the configuration information of current cell and common frequency adjacent cells, and performing multi-cell channel estimation for the cross-slot and obtaining the result of channel estimation; the BS determining the code channel configuration information of common frequency adjacent cells, and grouping the determined code channel, and determining the common frequency adjacent cells’ code channels which take part in joint detection; the BS obtaining detection data according to multi-cell channel eatimation result, each determined BS’s code channels which take part in joint detection. By use of the invention, cross-slot interference between the BS sending downlink signal and the BS receiving uplink signal in the corresponding slot is decreased, and sysytem capability and stability is improved.
Full Text The present invention relates to a time-slotted CDMA system, particularly to a method for suppressing cross-slot interference in a time-slotted CDMA system.
