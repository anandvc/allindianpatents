Title of Invention

A PUSH TYPE BATTERY TERMINAL POST CONNECTOR FOR STORAGE BATTARIES
Abstract A NOVEL PUSH TYPE BATTERY TERMINAL POST CONNECTOR includes as battery terminal connectors and wire. Battery terminal post connector is quadrangle shape so, we can fix it on all diameter batteries terminal. Which is coated by unique shape with has two fringes to easily get fix or remove from the respective battery terminal. Each battery terminal post connector is connected with wire due to the molding or it is fitted with nut-bolt on folding wire battery terminal, so wire is not come out from the connected metal terminal. In this type battery terminal post connecter power lose is reduce and this type battery terminal post connecter has low corrosion, so it is more economical in all the industrial application.
