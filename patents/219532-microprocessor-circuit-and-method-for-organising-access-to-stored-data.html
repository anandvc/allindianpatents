Title of Invention

MICROPROCESSOR CIRCUIT AND METHOD FOR ORGANISING ACCESS TO STORED DATA.
Abstract TITLE: MICROPROCESSOR CIRCUIT AND METHOD FOR ORGANISING ACCESS TO STORED DATA. Microprocessor circuit and method for organizing access to data or programs stored in a memory, having at least one microprocessor, a memory for an operating system, and at least one memory for free programming with individual external programs, a plurality of memory areas with respective address spaces being provided in the memory for free programming, each address space being assigned an identifier (program ID), and having means which load the identifier (program ID) respectively assigned to a memory area into a frist auxiliary register in each case in the event of memory accesses prior to the addressing of the memory area and which load the identifier (program ID)of the addressed memory area into a second auxiliary register and which perform a comparison of the first and second auxiliary registers wherein each address space of a memory area is assigned at least one bit sequence (ACR) comprising access rights.
