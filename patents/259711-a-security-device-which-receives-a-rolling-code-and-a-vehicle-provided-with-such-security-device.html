Title of Invention

A SECURITY DEVICE WHICH RECEIVES A ROLLING CODE AND A VEHICLE PROVIDED WITH SUCH SECURITY DEVICE
Abstract [Problem] To provide a security deviceand a vehicle provided with a securty device which prevent the registrantion of an incomplete rolling code even when a battery voltage is lowered. [Means for Resouluton] A vehicle provided witha a securtiy device includes a kick pedal 19, a generator (ACG) 1 which is connected to the kick pedal 19, a power source circuit 41 which changes an outpur voltage of the ACG1 into a given drive voltage, a capacitor 42 which is charged with an output voltage of the ACG1, a memory (DDPROM) 45 which stores a rolling code, a CPU 44 which updates the rolling code, and an AD input circuit 47 which monitors a votage which represnts the drive voltage, wherein the CPU 44 stops the updating of the rolling code when the drive voltage assumes a value lower than given reference voltage.
