Title of Invention

METHOD FOR RECOVERING PASS CODES / ACCESS CODES TO RESUME AN USER AUTHENTICATION IN A MOBILE DEVICE
Abstract The invention relates to a method to retrieve access codes/pass codes for an application on a mobile device when the access codes are lost or forgotten. According to the method, when the application is activated for the first time, a user of the mobile device is asked to provide the access codes, a destination address. The access codes and the destination address are stored in the memory of the mobile device. If the user loses the access codes, the user is prompted to enter the address. If the entered address matches with the preset address I the codes are sent to the preset address and the user may resume accessing the application.
