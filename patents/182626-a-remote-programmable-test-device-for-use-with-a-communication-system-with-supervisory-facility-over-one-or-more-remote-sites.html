Title of Invention

A REMOTE PROGRAMMABLE TEST-DEVICE FOR USE WITH A COMMUNICATION SYSTEM WITH SUPERVISORY FACILITY OVER ONE OR MORE REMOTE SITES
Abstract A remote programmable test device for use with a communication system with supervisory facility over one or more remote sites, said device comprising a control processor with attendant random access memory RAM and input/output I/O interface unit coupled to a system bus and a pair of memory systems for respectively storing active and inactive quasi-redundant versions of the operating system firmware and for electronically modifying the functionality of measuring and test routine, thereby monitoring and testing network lines and subscriber termination equipment coupled thereto.
