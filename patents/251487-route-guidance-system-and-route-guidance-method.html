Title of Invention

ROUTE GUIDANCE SYSTEM AND ROUTE GUIDANCE METHOD
Abstract A route guidance system enabling a driver to accurately recognize an intersection to which they are guided. The route guidance system has a current position detection section, a route search processing section, and a guidance processing section. The guidance processing section has branch element calculation and processing means for calculating the number of branch elements from the position of a driver's vehicle to a guidance point and also has waypoint guidance and processing means for performing route guidance based on the number of the branch elements calculated. Since the number of the branch elements from the position of the driver's vehicle to the guidance point is calculated and the route guidance based on the number of the branch elements calculated are performed, the driver can recognize the guidance point only by counting the number of the branch points. The driver does not erroneously recognize the guidance point.
