Title of Invention

METHOD FOR ATTRIBUTE NEGOTIATION IN EVOLUTION-DATA OPTIMIZED (EVDO) COMMUNICATION SYSTEMS
Abstract The present invention to a method and system for attribute negotiation by attribute value grouping. The present invention has been provided to form several groups for the values of the attributes of a protocol/application subtypes. The group to be used for a protocol/application subtype is negotiated at the time of protocol/application subtype negotiation itself. This will avoid negotiating attribute values for attributes by each protocol/application subtype independently. Mechanisms have also been provided which allows the protocol/application subtype to change some value, which the protocol/application subtype is using corresponding to the negotiated group.
