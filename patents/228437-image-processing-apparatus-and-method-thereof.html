Title of Invention

IMAGE PROCESSING APPARATUS AND METHOD THEREOF
Abstract The present invention particularly relates to an image processing apparatus capable of removing a moving cloudiness contained in a defocused image. A region identification block (103) identifies a non-mixed region including a foreground region consisting of a foreground object component constituting a foreground object and a background region consisting of a background object component constituting a background object, or a mixed region where a foreground object component is mixed with a background object component. In accordance with the region identification result and the like, an isolation/cloudiness removing block (1503) isolates a foreground object component and a background object component from pixel data of the mixed region and performs processing to remove moving cloudiness from the isolated foreground object component in a batch mode. This can take into consideration a difference between a signal detected by an image pickup element and a real world is considered.
