Title of Invention

AN AIR TRAFFIC CONTROL SYSTEM FOR CONVEYING LOCATION OF AN OBJECT.
Abstract The invention relates to a DES algorithm-based encryption method. The invention can be used in communication systems for sending data packets between user devices or nodes which are connected to a network. The inventive method is characterised in that it consists in: generating different random keys (6) for each packet, which are used to encrypt each packet using the DES encryption algorithm; and sending the random key (6) in the heading of the packet, said key having been encrypted (9) using a more secure algorithm than the DES algorithm. In this way, the invention can be used to improve security relating to transmissions over a non-secure means to which the nodes and user devices of a telecommunication system are connected, using the inventive method instead of the DES encryption algorithm.
