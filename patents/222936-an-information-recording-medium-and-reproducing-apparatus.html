Title of Invention

AN INFORMATION RECORDING MEDIUM AND REPRODUCING APPARATUS
Abstract Apparatus for recording data on an information recording medium comprising means for dividing data inputted in a time series into a plurality of divided data series each forming one row of predetermined number of bytes. A plurality of rows farm a sector and plurality of the sectors form an error correction block. A code is added to each sector for its identification. Means provided for adding a synchronizing signal to each row. A synchronizing signal of the row containing a first position of the code enabling identification of the sector being different from a synchronizing signal of all rows in the sector not containing the first position of the code enabling identification. The synchronizing signal of the row containing the first position of the code enabling identification of a sector being the same as a synchronizing signal of a. row of another sector without the first position of a code for that sector. Means provided for recording data inputted and processed on the recording medium with a sequence on the data inputted in time series being kept unchanged.
